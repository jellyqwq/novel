<p align="center">后记</p>

各位读者大家好，我是渡航（Speakeasy）。好久不见的渡航。

每一天，我都过着工作满档的日子。

最近我发现，在平均睡眠时间两个小时的生活持续半年之后，泪腺也跟着变得异常发达。凌晨四点时一个人在房间里哭得唏哩哗啦，再精神奕奕地前往工作，我过着的正是这种莫名其妙的生活。

我并不是觉得哀伤，更没有为了什么事情感动。要说我哭是因为痛苦的话，事实又不是那样，只不过是有股想哭的冲动忽然涌上心头，更何况我对痛苦的日子其实早就麻痹了。

如果要为流泪这种行为本身找个理由，的确是可以找出几个理由。比方说，工作的痛苦、超不想工作的痛苦、明明在工作却被人逼着工作的痛苦、光之美少女说着“我绝不放弃！”勇于面对危机的感动、美味的白饭、沉浸在平凡的幸福中时接到噩梦般的电话，又或者单纯只是太想睡觉打了个大呵欠。

有时候，连哭的人也搞不懂自己为什么在哭。话说在前头，我最爱工作了！不过，要是有人逼我工作，我会把他们全部杀光！全部杀光光！

极端来说，正因为不明白理由与因果，我们才会思索起这样的行为。

泪眼朦胧的扭曲视野里，我们只是专注地凝视单纯的现象，或是某人隐藏在现象背后的思绪。尽管明知道不可能看得见，尽管心死地认为反正不可能知道，但还是想弄清楚，所以我们抹去泪水，咬紧了牙想找出真相。

本集《Qualidea Code 心灵代码》第三集描述的正是这样的故事。这个世界在你眼中是什么样子？答案或许与我不尽相同，但依然期盼你能从中找到些许的希望。

《Qualidea Code 心灵代码》里的故事正是所谓的共有世界，世界观与设定相同，内容则由大家自由发挥，创作出各自不同的故事。除了我以外，另外还有さがら総老师和橘公司老师共同参与，由我们三个人全力打造出整体的故事内容！因此，本作品另外还有几部前传。

神奈川篇《为了拯救世界的那一天─Qualidea Code─》（橘公司着）。

东京篇《那样的世界毁掉算了─Qualidea Code─》（さがら総着）。

千叶篇《世界什么的怎样都好─Qualidea Code─》（渡航着）。

除了这三个都市的故事，还有所有故事的前传《废柴与金币的库洛迪亚》（渡航＆さがら総着），JOHANNES～
