青生轻抚着夕浪的遗骸，再用另一只手碰触朝凪。

朝凪不懂她这么做是什么用意，但在从她身上感受到命气脉动的瞬间，他终于理解了。

八重垣青生的〈世界〉。影像的共有，心灵相通的〈世界〉。

藉由青生的〈世界〉，朝凪脑中浮现出夕浪爱离的身影。夕浪露出圣母般的温柔微笑，敞开双臂，拥抱着朝凪与青生。

“太好了，爱离──你有女儿了。”

朝凪摸着青生的头，同时抱住了夕浪的身体。

他们的头上，大量瓦砾砸落，支撑地面的躯体嘎吱作响，沙尘蔓延开来，轰声响遍四周。

弥漫的烟雾背后，父母与女儿消失了身影。

战祸留下了难以抹灭的痕迹，湾岸的水中都市现在俨然成了一座废墟。

战争结束，似乎也改变了人们对世界的看法。

彷佛融入这片祥和的景色，JOHANNES军的旗舰停靠在东京湾。

医疗室里，夜羽唤来朱雀和卡娜莉亚，慰劳他们的辛劳。

“辛苦了，接下来交给我们处理，你们可以回到真正的人类所在的地方。大家都在等待解放关东的英雄，当然还有你们的父母。”

也许是身体状态稳定下来，夜羽在病床上坐了起来。她的视线高度因此与朱雀相近，然而朱雀不着痕迹地别开了目光。

“真正的人类吗？我们依然拥有〈世界〉这项能力，真的可以称作‘人类’吗？”

“小壹！”

因为从朱雀不屑说出的这句话里面感受到剑拔弩张的气氛，卡娜莉亚急忙告诫他。

不过，夜羽没有谴责他的态度，这类的冷嘲热讽她已经习以为常。她轻柔微笑，开始开导他。

“这话简直是胡说八道，就像‘我比底下的人优秀’一样。这不过是个性而已。决定人类的是内心，你们无庸置疑都是我们的孩子。”

“…………”

夜羽的说法深深打动了朱雀。

决定人类的是内心──巧合的是，朝凪也说过同样的话。

既然他们说过一样的话，这话应该有相当高的可信度。虽然他不信任千种夜羽这个人，但她毕竟是为人父母，而且还是千种霞与千种明日叶的母亲。他们说话总是拐弯抹角或是语焉不详，不过传达出的都是真相──他记起了这件事。

于是他决定相信，相信朝凪留下的话。

朱雀改变想法。“我是开玩笑的。”他摇了摇头收回前言，然而，马上又继续说下去：

“可是，我们还不能离开这里。”

“为什么？”

“这里是我们的城市，就算要回去看我们的父母，也得等复兴这个地方之后再说。要是让这里维持在这样的惨状，怎么对得起东京首席的名号？”

“你大可不必再受UNKNOWN制定的名次束缚……”

夜羽说到一半，惊慌地按住了嘴。

“难不成我太不识相了吗？”

“啊哈哈哈！”

担任朱雀翻译的卡娜莉亚笑着给了肯定的答覆。朱雀耸耸肩道“我不知道”，难为情地转过头去。

在他的视线前方，有个人正偷偷摸摸地窥探室内的状况。不过，那个人像是再也按捺不住，迈开脚步踏进室内。

“嫣妈。”

“哎呀，你来啦。”

明日叶走了进来，她的语气兴奋，脚步也很雀跃。千种明日叶居然会有这么亲昵的表现，朱雀很惊讶她竟有这意外的一面。

“哥哥，快点啦！”

“好啦、好啦。”

接着，霞慢吞吞地走进室内。

他的神情和平常一样，分不出是生气还是在笑，而且也许是多心了，他好像有点驼背。

朱雀与卡娜莉亚看着他们，一起离开了医疗室。

朱雀没那么迟钝，打扰他们一家团圆。

只是在离去前，他对霞投以挖苦的视线。霞啧了一声，微微向他点了个头。朱雀心想他这动作也许是在表达谢意，开心地笑着走了出去。

霞没有答谢的意思，他其实只是害羞。同龄的朋友看见他和母亲在一起的样子，实在令人觉得很难为情。

等朱雀他们离开后，霞总算在夜羽的病床旁边坐了下来，而明日叶正黏着夜羽撒娇。这情景让我想到那只猫不知道为什么老爱黏着妈妈，可是只要我一靠近就逃走……他想起了很久以前养的猫。

“你回来啦……”

“……嗯，我回来了。”

夜羽疼爱地摸着明日叶的头，明日叶轻闭上双眼，深情地喃喃开口。简短的对话彷佛融解了长年分离的时光。霞眯起眼睛看着这对母女时，夜羽朝霞微微一笑。

“霞，你也回来了呢。”

“嗯。啊，辛苦了。”

霞有些犹豫要说什么，最后说出了这句话来。夜羽听见后，气呼呼地鼓起脸颊。

“唉，我不是教过你要认真打招呼吗……”

“不、不，在我们的组织里面、适就是正式的招呼语。不管早晩都是‘早安’，从一天的早上工作到另一天的早上，回家的时候就说‘辛苦了’。”

霞随口乱掰，听得夜羽摸不着头绪。接着，她问起躺在大腿上的明日叶：

“有这回事吗？”

“那是哥哥才这样。真要说起来，只有哥哥是这样。”

“是啊……只有我在工作嘛……”

霞的嘟囔声里带着哀愁。日以继夜埋头忙着战后处理的人自然是霞。也许是怜悯他的遭遇，夜羽轻咳了一声。

“不过，你现在没必要在那种过劳的环境下工作了吧？所以说，欢迎来到JOHANNES军！”

“那里好像会是一家更可怕的黑心企业……再说，拜托别顺势要我继承职位好吗？”

“这里可是提供年轻人活跃的机会，世代交替管道畅通，每个人都可以发光发热的温馨战场喔？”

“这种话在千叶也很常说……”

夜羽的这番好听话听得明日叶不禁畏怯，或许是明日叶胆怯的反应让夜羽内心受到了伤害，她的语气变得有些低落。

“可、可是，温馨的大家庭是事实啊……”

“那只是家族经营吧……”

霞说着，疲惫地叹了口气。

霞明白夜羽的意思。就算解释得口沫横飞，她终归是舍不得和孩子分开──霞下了这样的结论。

既然这样，霞有自己应该做的事。

“要说一家人的话，还少了一个人……我去找回来。”

““什么？””

两人异口同声地叫了出来。

在能够悠然眺望东京湾海景的海滨公园遗迹里，一座简朴的石碑坐镇在小山丘上。

那正是一座墓碑。

在这里，有两个人献上了花束，以及老旧的怀表。

“这座坟墓底下什么都没有呢。”

舞姬忽而望向脚边。

“是啊，可是没东西，不代表没有意义。”

“……嗯。”

在萤的安慰下，舞姬抬起头来。

她实在难以接受事实。

天空与大海都美得讽刺，风无情地吹过两人之间。

然而，这阵风同时也带来了小孩子兴奋的嬉闹声。

往小山丘底下一瞧，副司令官他们正带着从冷冻睡眠中醒来的孩子散步。

多么平和且慈爱的景象啊。

舞姬百感交集，神情不再严肃。

“爱离小姐她……她对我们很温柔呢。”

“是啊。”

“她就像我们真的母亲一样。”

“是啊……”

“不过，真伤脑筋，我还是没搞懂她以前说的那些话是什么意思。”

她想起很久以前，自己年纪还很小的时候，正好就在像今天这样风和日丽的公园里和夕浪聊天。

夕浪说过，想要自己的孩子。

她微笑着说，完全不同的两个人相遇、诞生出新生命、让生命延续下去──这个世界的生物很美妙。

自己喜欢的每个人都在这个世界，舞姬能够明白她觉得美妙的心情。

但是，相遇、诞生、延续生命……这些事她连一件也搞不懂。在舞姬心里，世界是更模糊的样貌。

“如果以后能明白就好了……”

“嗯，是啊。”

受到萤温暖的支持，舞姬点了下头。

她希望自己能够确实理解夕浪话里的意思，并且认同她的想法。

虽然不知道自己有没有资格说这种话，不过她的确很喜欢夕浪。

因为期许自己也能成为那样的大人──

“……小萤。”

“什么事，公主？”

“小孩子要怎么制造出来？”

“！！！？”

萤惊讶地咳嗽起来。“你没事吧！？小萤！”舞姬轻抚她的背部。

自己说了那么奇怪的话吗？她以为如果要成为像夕浪那样的母亲，首先必须知道这件事……看来她离长大成人还很遥远。

舞姬不禁烦恼，萤则是烦闷不已。这时，热闹的脚步声往这里接近。

“嗨。”

慵懒的呼唤声响起，舞姬转过头后，随即绽放出灿烂的笑容。

“明日叶！霞、朱雀还有卡娜莉亚！”

熟悉的脸孔齐聚一堂，一行人往山丘爬了上来。他们应该是来扫墓的吧。明日叶捧着一束花，朱雀也提着一纸袋的零食。

“我、我只是碰巧经过而已。”

没有人问起这件事，朱雀却兀自辩驳了起来，把头甩到一边去。“真是不老实。”卡娜莉亚在朱雀身边笑着，舞姬也跟着笑了起来。

所有人双手合十，献上默祷。

寂静的时光里，他们暗自道着内心的思绪。

花束随风摇曳，朱雀轻吁了一口气。

“──你们接下来打算怎么办？我会尽自己身为东京首席的职责。”

“我们也是一样。重振神奈川，然后、然后……有好多事情要做！”

舞姬高举起拳头，气势十足地吆喝，萤也往她靠了过去。霞你呢？舞姬用视线询问起霞。

“这里已经没有我的事，再说我也独立了，可以做自己想做的事。”霞轻轻耸了耸肩。“要我继承实在太麻烦了……之后肯定会有人跑来找我问东问西，我哪记得啊，资料也早就作废了……”他嘀咕着抱怨了起来。

明日叶往霞瞥了一眼──我也要做自己喜欢的事。她把手中的机车钥匙晃得当啷作响。不管是喜欢还是想做的事，她心里都有了底。

朱雀听着霞的玩笑，再看明日叶用指尖摇着机车钥匙的模样，忽然有种想法。

霞口中的‘这里’指的是哪里？恐怕不是指他现在站的这个地方，而是更大范围的‘这里’。

充分理解他话里的意思之后，朱雀朝霞的背影蹙起了眉间。

“你老是这个样子。只根据自己的道理前进，完全不考虑留下的人的心情。”

“小壹！？”

卡娜莉亚注意到朱雀的口气火爆，整个人惊慌失措。不过，霞只是笑了一声，把上半身往他转了过去，脸上浮现和平常一样扬起单边嘴角的嘲讽笑容。

“什么？这是在谈分手吗？这种话只会造成我的困扰，再说也很恶心。”

“……恶心的人是你。”

朱雀嘴上这么回应他的奚落，嘴角沉稳地放松了下来。

阳光穿过层层白云，洒落在两人之间。争辩的气氛消散，只有闲聊带来的自在微笑。

卡娜莉亚不禁感到错愕，舞姬和萤强忍笑意，明日叶则笑嘻嘻地看着霞。

霞心想，这样就行了。

这种形式最适合他们。用卖弄玄虚的答案回答真诚的话语，但意思又能确实传达到对方心里。

所以说，这样就可以了。

霞别过头，直接迈步走去。临走前，他只抛下了一句话──

“……再见。”

“再见。”

回应从背后传了过来。

他们互道再见，连目送时也没看彼此一眼。

分离并未让他们感到惋惜。

因为他们一直是水火不容的两人。

所以──

他们背对彼此坦然地微笑，各自走上自己的路。
