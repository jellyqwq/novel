<p align="center">后记</p>

各位读者大家好，我是渡航（Speakeasy）。与各位许久不见的渡航。

现在的时期正值盛夏，大家过得都好吗？

大家有去海边吗？还是去游泳池呢？或是在可以避暑的地方度假？也有可能在度假胜地开趴吧。我则是在工作，就算不是夏天，我也在工作。

夏天这个词给人的印象大多是开放或是动感的，像是豪放又快活的疯狂派对动物。

此外，夏天让人联想到蚊香的香味，也有人会讲出这种文青又帅气的话来，虽然我也不清楚蚊香这种东西算不算得上帅气。

夏天待在冷气房里，在某种意义上一点也没有夏天的感觉，窗户不打开也不会有蚊子跑进来，也有这种万年室内派的存在。每个人对夏天的看法各有不同，这也可以看出每个人的特性和性格。

就像这样的感觉，在同一个世界里，有各种不同的事物在活动，发展出故事。因为一样所以有趣、因为不同所以好玩，如果可以像这样在这世界玩乐，一定很有意思。

本集为《Qualidea Code 心灵代码》第一集。

《Qualidea Code 心灵代码》里的故事正是所谓的共有世界，世界观与设定相同，内容则由大家自由发挥，创作出各自不同的故事。

除了我以外，另外还有さがら総老师和橘公司老师参与这个计画。

本书由本人渡航执笔，但是故事是大家共同创作！
