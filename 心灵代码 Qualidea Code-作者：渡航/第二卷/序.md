──小姬，你还好吧？

「嗯，我不要紧。」

──这样啊，小姬果然很坚强呢。

我在门的另一头，看著公主与亲戚交谈的背影。

我和公主的年纪都还小，但也已经大到足以明白丧礼代表什么意思。

亲人的逝世，告别的仪式──

是公主再也不能见到自己挚爱爷爷的证明。

所以那个人才会这么说吧──说她很坚强。

公主确实很强，这一点无庸置疑。

虽然体型比我娇小，但是她的内心比我还要坚强。

不过，那和大家所说的坚强好像又有点不一样。

看著公主将爷爷最重视的大衣与怀表紧握在手中的模样，我再次有这样的感受。

并非没有对爷爷的过世感到伤心，可是她在这种时候绝对不会让眼泪流下来。

因为要是她哭了，她身边的人会更难过。

不管是爸爸妈妈还是那些亲戚，所有人的心情都会因此变得哀伤。

为了不让伤心的情绪蔓延开来，她必须强迫自己『坚强』起来。

所以，我动也不动地站在原地。

我强抑住想马上往公主背后冲过去的心情，始终待在那个地方。

这段时间顶多只有十分钟左右，而我所做的事只是站在原地。

不过，这却是比其他痛苦的时候，更来得难受与心痛的一段时间。

等所有人都从公主身边离开后，我这才往她身边走去。

──公主。

「……！」

听见我呼唤她的名字后，她惊讶地把头转了过来。

「……、……」

她什么话也没说，只是用力抓住我。

泪水在丧服胸口处缓慢渲染开来。

嗯，没关系的，公主。

有我在，我会永远和你一起。

我们一起──变得更强吧。
