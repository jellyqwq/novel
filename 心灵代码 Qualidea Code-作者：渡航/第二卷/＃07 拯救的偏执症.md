废墟荒芜的地面上，一辆大型摩托车轰鸣著一路往北前进。

「哇啊，情况太恶劣了吧？居然有〈UNKNOWN〉从天而降。」

「啊……真的很糟糕。」

负责驾驶的明日叶望著前方的东京都市上空发出惊呼声，坐在边车上的霞点头回应。

突如其来的，有大批〈UNKNOWN〉出现在东京。过去，〈UNKNOWN〉只会透过东京湾上空的传送门出现。管理局经常宣导，南关东一代有防护罩笼罩。防护罩可谓万全的防空设施，只是因为范围没有扩及海上，霞他们才需要防守以跨海大桥为首的要冲。

然而，这个常识如今已被轻易瓦解。

尽管早就知道这样的状况，一旦当面遇上了，实在令人毛骨悚然。

「就是说啊！糟糕糟糕！真是太糟糕了！」

尽管是毛骨悚然的景象，明日叶却没有产生危机感或是焦躁，而是兴奋不已、无比雀跃。

霞侧眼看著她，不得不感到纳闷。

「我说你啊？你是不是搞错意思啦？──明日叶，前面！」

霞惊慌失措地大叫，像是代替明日叶的角色。

出没在新宿上空的〈UNKNOWN〉疑似进攻到了郊外，埋伏在废墟阴影处的欧格级倾巢而出，纷纷杀向霞他们的摩托车。

「我知道啦！没想到有这么盛大的欢迎！」

四面八方遭到〈UNKNOWN〉包围，明日叶不仅不胆怯，甚至把油门踩到底。她蛮横地操纵握把，华丽地在敌阵中穿梭。

为了不被车子甩下去，霞死命抓住边车，试图谴责明日叶的野蛮行径。只是他一眼就看出来，明日叶的战意破表，已经到了什么话都听不进去的境界。

明日叶甚至没有握住摩托车的握把，手里取而代之握住的是两把枪。

她好战地扬起嘴角，撒出冰炎的子弹。

尽管乍看之下像是随便射击，每一发子弹却都准确地贯穿了没有面貌的〈UNKNOWN〉。

没有射中目标的子弹火焰轰垮废墟，原本以为会撒下大量瓦砾，结果又有子弹击中瓦砾。

接著，大量洒落的瓦砾冻结在一起，临时在眼前筑起一座上倾的斜坡。

明日叶收起两把手枪，再次握住摩托车握把，骑著车冲上延伸向天际的冰坡。

「呀喝──！」

「喔喔喔！？」

从冰坡的顶端，摩托车循著惯性冲了出去。

有数秒钟的时间，厚重的铁马确实飞到了半空中。

霞倒抽著气紧抓住边车，臀部感觉到剧烈的冲击，痛得让他不自觉揉了揉自己的屁股。

不过，摩托车冲上冰坡的力道没有减弱，在高架的高速公路上著地后，直接冲了出去。

「欸！你别太乱来！」

「哈哈！再来再来！」

「完全没在听我说话……嗯？」

霞回头看向来时的方向，发现有一具〈UNKNOWN〉固执地追了上来。霞一看见，马上啐了一声。

他代替不只没有注意周围的状况，连声音也听不见的明日叶举起步枪，击落了那具〈UNKNOWN〉。

霞吁了口气，不过他也只有这一瞬间能放松心情。

「……嗯？」

他不经意地一瞧，发现只有一旁明日叶骑的摩托车缓慢加速，往前面冲了出去。

他原本还纳闷著是怎么回事，然后才注意到边车与摩托车连结的地方断裂了。

「好，继续前进──！」

「拜托不要用神奈川标准来判断好吗？你们根本就不是人。」

忍不住苦笑的青生关心起霞，霞像是获得救赎，嗓音不禁颤抖。

三十年前的寻常日常生活，现在就算有钱也买不到。

敌人一再来袭，没有时间可以让心情放松下来。

与家人分隔两地，想要有人可以依赖或是哭泣的时候，他们都不在身旁。

人心远比自己以为的还要脆弱。

遇上这种时候，人类总会忍不住当场蹲下来。

不过，这个样子不行。

世界随时在动，要是停下脚步，马上会被甩到后头。

所以说──需要那样的存在。

让大家能抬起头来的话语。

让大家在抬起头的时候，在前进道路上有个能追随的背影。

即使眼前的敌人再怎么强大。

即使伙伴在战斗中一个一个倒下。

──即使长久以来相信的其实是个虚假的世界。

大家需要一个不会迷惘、迟疑，也不会动摇的背影。

咳咳，各位，让我们前进吧。

战场不再是我们知道的样子，拋开你们过去所有的常识。

那是个假象的世界，敌人不是敌人，伙伴也不是伙伴。

然而，这并不会改变我们该做的事情。

我们的目的不是与敌人作战，我们只是要去取回三十年前忘记带走的东西。

事情──就是这么简单。

所以说，我们走吧，小萤。

──为了拯救世界的那一天。
