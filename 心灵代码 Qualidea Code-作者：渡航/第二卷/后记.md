<p align="center">后记</p>

各位读者大家好，我是渡航（Speakeasy）。和大家好久不见的渡航。

不知不觉，已经到了耶诞……真要说起来是到了年底。果真是光阴似箭，岁月如梭啊。尤其在即将年满三十岁的最近，时间更是流逝得比学生时期还要飞快。

时间面前人人平等，不论是孩童、国中生、女高中生、社畜还是老人的一年，都是同样一年的时间，只是每个人对时间的感觉简直完全不同。

在开始工作之后，我对月历的感觉也改变了。原本引颈期盼的长假──像是暑假、年底和过年，现在在我心中只有无比的憎恨。

黄金周提前送印、暑休提前送印、年底提前送印……因为需要提前送印的情形太多，简直都要产生出新的信仰了。

顺带一提，各出版社里面存在破坏这些提前送印时间的渡航流程，而且这样的流程惹来了许多憎恨。我觉得很难受，也很过意不去，从罪恶感里面好像要产生出新的信仰来了。

总之，因为年龄、立场和头衔，以及个人所处环境与培养出来的价值观不同，看见的世界也不一样。

在他们眼里，那个世界现在不知道是什么样子。

本集即是《Qualidea Code 心灵代码》第二集。

『Qualidea Code』的故事正是所谓的共有世界，世界观与设定相同，内容则由大家自由发挥，创作出各自不同的故事。除了我以外，另外还有さがら総老师和橘公司老师参与这个计画。

本书由本人渡航执笔，但是故事是大家共同创作！共同创作实在是一件很美妙、很棒，也很快乐的一件事！另外，本书还有几本前传。
