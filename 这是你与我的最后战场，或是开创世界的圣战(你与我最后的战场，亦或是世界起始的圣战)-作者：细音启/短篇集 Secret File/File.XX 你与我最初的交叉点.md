1

「等一等！等等我啊、师父！」

一边呼出白色吐息——

黑发少年伊斯卡一边追赶着走在前方的男人的背影。

夕阳染红的大陆铁道的主要站终点站。

在旅行者络绎不绝的道路上，无论伊斯卡如何加快步伐地追赶也难以缩短二人之间的距离，其原因想必是二者步幅的不同。

相对于只有十一岁的年幼少年伊斯卡，他称呼为师父的男人的身高则将近一百九十公分。

「真是的，师父总是这样把我丢在一边！」

「…………」

突然停下脚步的男人回首看向了伊斯卡。

「丢下？谁把谁丢下？」

「你！把我！丢下」

「…………」

「难道你没有注意到？」

「我正在想点事儿」

哈啊……

听到完全没有羞愧之心的师父的回答，伊斯卡只能无奈地耸了耸肩。

这个男人总是这样子。

琢磨不定、随心所欲，总是心不在焉，本以为他会说些什么的时候却又常是一副无精打采的样子——

但他也是帝国最强的剑士。

克罗斯威尔•涅斯•利比盖特。

毫无赘肉的修长身段、黑发、身披长款风衣。

曾经身为使徒圣笔头时的异名好像是「黑钢的剑奴」，但本人却很少谈起当时的事情。

他本人曾说，不是不想说，而是说起来很麻烦。

「顺便问一下，你在考虑什么事情啊？」

「考虑关于这辆列车的事情」

师父克罗斯威尔盯着的是乘车处并排停着的外形粗犷全身涂黑的快速列车。

它们将从这个主要车站出发，驶往世界各地。

「十五分钟之后，列车就要出发了。我们也要乘坐在里面」

「没错」

「我在想要是凶恶的犯罪组织也偶然在这趟列车上，他们暴动的时候该怎么压制他们」

「这种未来的可能性得多低啊！？」

「要是列车行进中，突然天降陨石的话——」

「请想一点更具现实性的事情！？」

「预测未来可是很重要的」

这样回答的师父竟然满脸认真的表情。

「假设一切不幸及不巧的场合，其中肯定有几种必定会发生。无论是否身在战场，亦或是除你之外的伙伴」

「……好的」

明明开始于摸不着头脑的事情，可最后竟用「感觉是很正常的事情」来做总结。这也是和师父谈话时的必然情况。

就在此时——

『前往维尔共和国的特别高速列车不久就要发车了。请持票的旅客静候乘车』

「话说回来，师父？」

一边倾听着广播，伊斯卡突然抬头看向了师父。

「我们为什么要做列车啊？」

话说回来，伊斯卡甚至不知道现在究竟是旅行还是远征。

就在昨天，因为师父突然来了一句「要出门了」，所以伊斯卡才做好了准备，然就像往常一样师父几乎不会告诉自己他的目的。

「留守已经交给烬了」

「……我们是去帝国外修行吗？」

「和修行没任何关系」

竟然不是修行？

对于每天都陷于残酷修行的伊斯卡来说，他已经做好了在本次远行的目的地有无比残酷的修行在等着自己的觉悟。

「我们去帝国外是为了做什么啊？」

「为了了解帝国之外的情况」

「了解帝国之外的情况是为了什么呢？」

「…………」

帝国最强的剑士抬头看向了主要驿站的天花板。

「因为你还不了解魔女究竟为何」

「……我还是了解一些的」

恐怕没有帝国人不知道「魔女」。

寄宿着星灵这一未解析能量的「舍弃为人者」。魔女即是操纵星灵之力的恐怖存在。

——凶暴且具有攻击性，以及憎恨帝国之人。

伊斯卡对魔女便是这种印象。

为什么要说是印象，因为伊斯卡本身并未和魔女有过交谈。

这些全是帝国内口耳相传的情报。

「我并没有说你对魔女的印象是错的，只是，这并非魔女的全貌」

师父环视着在主要驿站内来来往往的行人，说道。

「帝国内流传的魔女的轶事，只是像大魔女涅比里斯那样极少的例外引起的事件。百分之九十的魔女都和普通人类没什么两样。伊斯卡，看到在这个主要驿站内来往的人们你有何想法？」

「……看起来只是一些普通人」

乘坐列车的商人以及带着家人的旅客们。

在伊斯卡眼里，他们只是「普通的人类」。

「从统计上来看，他们中肯定混入了魔人或魔女。但他们并没有和帝国人有丝毫不同吧，在你眼中他们真的野蛮吗？」

「不」

「到头来这也是真实。无论是帝国内口耳相传的轶事，还是你现在眼中所见的光景，都要牢记于心」

「…………明白了」

间隔了短暂的沉默后，伊斯卡点了点头。

魔女可怖——

虽然正在努力理解师父教导自己的话，但对于帝国内土生土长的伊斯卡来说，老实说难以抹去这一印象才是自己的真心所在。

「总有一天你会明白的，这次远行也是为了这一点」

师父迈步向前。

就在他准备走向列车的乘车处时。

「嗯？」

师父胸前的通信机鸣响了起来。

扫了一眼上面显示的名字后，师父竟然很罕见地咂了咂舌。

「……麻烦的家伙联络我了呢」

「师父，是谁啊？烬吗？」

「不巧，是别人。伊斯卡，你先乘车。……有什么事啊，尤梅隆根，别在这种时候推给我一些杂活啊」

一边和某人通话，师父一边朝远处走去。

像是不愿被周围人听到一样，师父一直走到了乘车处的角落。

「师父？真是的，我就先走了哦」

伊斯卡乘上了前往维尔共和国的列车。

见到窗边的自由座位还空着两个人的席位，于是伊斯卡占下了这里。

「真是的，师父打电话竟然会这么久，真让人意外。不知道还能不能赶上」

还有五分钟列车便要出发了。

背靠着座位，伊斯卡心不在焉地朝着车窗外望去。

「……真慢呢， 师父」

『前往遗迹都市格拉芙的快速列车就要发车了。请持票旅客静候乘车』

「伊斯卡？喂，伊斯卡」

广播响起，伊斯卡的师父克罗斯威尔扫视着车内。

不见弟子的踪影。

仅算上自由席位也有数个车厢。找起来自然很是辛苦，但自己都这样呼唤他的名字了竟然还未出现，真是罕见。

「难道为了等我还在列车外面吗？」

以防万一先离开列车吧。

就在此时——

就在克罗斯威尔来到列车外的时候，停在附近乘车处的高速列车发动了。

克罗斯威尔不禁目送着这辆列车离去。

「嗞！伊斯卡……！？」

克罗斯威尔突然瞪大了双眼。

这是前往维尔共和国的列车。而就在车窗内，和伊斯卡极其相似的黑发少年正坐在车内。

「伊斯卡！」

虽然克罗斯威尔慌慌张张大叫着伊斯卡，但为时已晚。

弟子乘坐的快速列车，正朝着远方疾驰而去。

「……那家伙」

手扶额头，克罗斯威尔无奈地大声叹着气。

「本以为假定了各种不妙的场合，可这种情况真的在意料之外」

那个笨蛋弟子。

想必他做梦也没想到自己会搞错列车。

2

「……真是慢啊，师父」

伊斯卡强耐性子。

已经等师父等了一个多小时。窗外的景色也早就离开了主要驿站，切换成了宽阔无垠的荒野。

「啊，我明白了」

面对突然掠过脑海的可能，伊斯卡猛地从座位上起身。

「师父真是的，肯定又在为难自己。那么他究竟躲在哪里了呢？」

随心所欲地扔给自己难以这一点伊斯卡也已经习惯了。

这次也是如此。之所以不现身，是因为想要我主动找到他。

「真是的……师父，你藏起来也是没用的哦。反正这种列车也没多少能藏身的地方！」

伊斯卡兴致冲冲地迈出了步伐。

从自由席位车厢走向指定席位车厢，一边确认乘客的样貌，一边朝着里面的车厢走去。

在这个瞬间。

轻轻飘动的金发轻触到了伊斯卡的鼻尖。

擦肩而过。

「…………」

伊斯卡无意识地转过脸来，发现那里站着一位有着炫目金发的少女。

年龄大概是十一或十二。

大概和自己同龄。如高贵人偶般端正可爱的面容，身上穿着的衣服也散发着高级品质的清秀感。

和穿了许久已经走样了的自己的衣服完全不同。

这位少女——

保持着擦肩而过的姿势停下了脚步，并死死地盯向自己。

「…………」

「…………」

她是谁呢？

难道在怀疑自己？这里是指定席位所在车厢，她的视线也许在暗示「只能买自由席位车票的穷人不要进来」。

在她叫来乘务员之前还是赶紧离开吧。

做出这种判断的伊斯卡转身迈出了步伐。毕竟自己现在还必须得找师父呢。

黑发少年迅速转身背对自己离开了。

不等自己出声便朝着里面走去了。本来就只是擦肩而过，他肯定有要事要去别的车厢。

「……离开了呢」

就在目送着他的背影远去直至消失不见的时候，后面传来了脚步声。

「爱丽丝大人」

「…………」

「爱丽丝大人，真是的，这样可不行哦。不要随便离开座位啊」

「听我说！恋莲！」

爱丽丝突然转身抱住了她称呼为「恋莲」的成年女性。

二人并非母女。

二人的关系是，主从。

魔女乐园「涅比里斯皇厅」的王女，以及其随身护卫。

「爱丽丝大人，你可让我好担心呢」

恋莲抚摸着紧抱住自己的爱丽丝的脑袋。

她戴着变装用的眼镜。她之所以身穿冬款厚重大衣，是为了在衣服内侧藏各种武器。

「真是的，爱丽丝大人，每次我一不注意，您就不见踪影」

「呐，恋莲！比起这些！」

然而身为当事人的爱丽丝却完全没听进恋莲的话。看来这个「大发现」已经占据了她的脑海。

「有个男孩子！」

「什么？」

「有个和我同年龄的男孩子。刚刚才从我眼前经过！」

「……啊，原来如此啊」

看着兴奋不已的爱丽丝，身为护卫的恋莲嘴角不禁柔和了下来。

爱丽丝——爱丽丝莉泽•卢•涅比里斯9世身边并没有同年级的男孩子。

全是一些年龄相仿的女孩子。

对于接受专属教师指导以代替上学的爱丽丝来说，并没有像平常女孩子那样在学校内和男孩子交谈的机会。

所以，和男孩子擦肩而过对她来说也是极为罕见的经验。

「呐、恋莲，我偷偷地看向了那个男孩子。结果，男孩子也突然转过身来看向了我！」

「毕竟爱丽丝大人很可爱呀」

「……是不是主动搭话更好呢」

「这点就不敢苟同了。好了，爱丽丝大人来这边」

身为护卫的恋莲朝爱丽丝招了招手。

自己前后左右的席位上并没有人，这是因为恋莲为了避开他人而买下了这些座位。

「爱丽丝大人的身份要是万一走漏了风声，可就麻烦了」

恋莲小声地在爱丽丝耳边叮嘱到。

「现在已经远离车站，护卫也只有我一个人，要是被帝国军发现了可就棘手了」

这辆列车正朝着维尔共和国驶去。

虽然是一个和帝国并无关系的中立国，但帝国军在世界各地配置了谍报机关这一点已经是众所周知的事实。

「被称为魔女而且还在街上被追赶，对爱丽丝大人来说也很难受吧？」

「没事的，恋莲，我的星灵可是很厉害的！」

抛开恋莲的不安，爱丽丝自信满满地挺起了胸膛。

「帝国军也只需我的星灵的一发攻击。女王大人也夸奖过我呢」

「嗯。但女王大人也这样说过哦，「爱丽丝，直到可以完美制御为止，都不可以使用你的星灵术」」

「……的确这样说过」

「制御花费时间并不是值得可耻的事情。相反，这是爱丽丝大人的星灵无比强大的证据」

王女爱丽丝的星灵极其强大，因此制御起来也相应的困难。

万一，和帝国军发生战斗的话——

爱丽丝的星灵术一旦爆发，不仅是帝国军，恐怕连其附近的大楼及无关人士也会被卷入过于强大的冷气之中。

「虽然不如帝国军那么明显，但在中立国内也有害怕我们星灵使的人。要是爱丽丝大人的术伤到了他们的话……」

「……会被人畏惧？」

「是的。我国也会被世界所孤立。我们星灵使是禁止在战场之外的地方使用星灵术的，特别是街上」

「……明白了」

星灵之力是把双刃剑，这点爱丽丝也深有体会。

就在星灵术刚刚觉醒的时候——

偷偷将地面稍微冻住的爱丽丝的「天真」，结果却让宽阔的王宫中庭整个化作了冰雕。

中庭内停着的大型车辆以及摩托也都化作了一团冰块。

万幸并未出现牺牲者，然而这幅光景却让爱丽丝同恐怖一同知晓了星灵术并不是方便的力量这个事实。

「……在列车到达之前可以读书吗？」

「这样最好」

「……能时不时在列车内散步吗？」

「希望您尽量不要。虽然明白刚才和男孩子擦肩而过引起了您对同年龄孩子的好奇心」

一边告诫着爱丽丝，恋莲一边露出了淡淡的苦笑。

「话说回来，爱丽丝大人也有需要搞好关系的同年龄的孩子哦。虽然男孩子会有各种问题，但女孩子就没任何问题了」

「……你说的是，谁呢？」

「我的侄女，是个小爱丽丝大人一岁的女孩子哦。名字叫燐，为了侍奉卢家，从小时候起就一直热衷于学习呢」

「燐？」

「嗯。那孩子也希望可以早日结束训练，尽快侍奉爱丽丝大人呢——」

恋莲的声音突然消失。

刺耳的紧急警报声同时在整辆列车内响起。

「怎么回事！？」

恋莲立刻从座位上起身。

之所以会把手伸进衣服内侧，想必是为了能立刻取出武器。

嘈杂声四起。

坐在远离爱丽丝和恋莲的其他指定席位上的乘客们，也因为突然响起的紧急警报而深感不安地站起身来。

「恋莲，这是在王宫内也听到过的声音呢」

「是的呢，爱丽丝大人。但是王宫内的紧急警报是假定帝国军侵入时避难训练使用的。可现在的毫无疑问是真正的紧急警报声」

「……帝国军？」

「不对。不太可能是因为他们」

帝国军盯上爱丽丝而发起的袭击？

不可能。这个大陆铁道可是世界协定的非武装地带。帝国军也不会这么简单就打破协定。

既然如此，这个紧急警报声究竟是因为什么呢？

『前往维尔共和国的第三号列车中的各位乘客，紧紧通知。本列车现在虽在伽拉德荒野内径直朝东方前进，但——』

车内广播。

包括爱丽丝和恋莲在内的乘客们一起安静了下来。

『被称为恐龙的大型食肉动物正在接近本列车。猜想应该是栖息在荒野遥远南方的恐龙北上至此』

「……恐龙！？」

某人的怒鸣声响起。

秘境内栖息着的大型食肉动物统称为「徊兽」，恐龙便是其代表。

广泛分布在大陆各地，凶猛且好战。有着袭击眼前一切运动物体的习性。

这样的恐龙群正袭向这辆列车？

『请各位放心。为防止这种事态，本列车配备了优秀的猎人。请大家冷静——』

轰鸣遮蔽了广播。

窗玻璃的破碎声响起。接着某物冲撞列车的冲击游走，最后传到爱丽丝耳中的是凶猛野兽的咆哮。

3

窗玻璃粉碎。

紧接着，伊斯卡身后的墙壁伴随着钝音凹陷了下去。

「呀啊！？」

「是恐龙！它们撞上来了！？」

这场骚动中可由不得自己冷静。

毕竟自己所在的车厢的玻璃已碎，恐龙锋利的爪子正在暗夜中若隐若现。

列车已经被恐龙群包围了。

「猎人呢！？猎人在哪里啊！？」

「朝里走！向对面的车厢里逃！」

乘客们争先恐后地跑起。

完全来不及拿行李，一同挤向附近的车厢避难。

——回过神来。

竟然只有自己伊斯卡一人还留在车厢内。

「……恐龙」

就在伊斯卡呢喃的同时，足以刺破鼓膜的枪声响起。

大概是猎人们的机关枪发出来的。玻璃碎裂的车窗对面，的确可以看到花火四散。

但是——

黑夜内，全长五米的大型食肉动物的影子仍旧在晃动。

毫无作用。

承受了子弹造成的疼痛，恐龙反倒更加狂暴了。

「……这样下去，可不行啊！」

它们侵入车厢内也只是时间问题了。

面对这股恶寒，伊斯卡脑海中浮现的不是猎人，而是应该在列车内某处的帝国最强剑士。

「师父，都这个时候了你在做什么啊……！」
