<p align="center">后记</p>

感谢您本次入手『被卷入了勇者召唤事件却发现异世界很和平』第十一卷。

这次在结尾给魔王篇开了个头，这也是中段的一个关键点。下一卷肯定也很受严肃前辈的期待吧。

这次的新角色有克里斯的母亲莉莉姆，这一角色在 WEB 版要到六王祭才登场，这次把登场提前了。不如说，内幕是六王祭登场的新角色太多了，所以才先像现在这样一点一点提前放出。

关于下一卷围绕魔王菲尔的战斗，由于提前登场的角色影响，内容应该会和 WEB 版有所不同。新写的部分会比较多，我觉得应该会有点辛苦，希望你们能怀着期待等着吧。

魔王篇结束后，终于该到六王祭篇了。后者也会比较长，不过几乎没有严肃情节，基本是快人和女朋友们享受祭典的感觉，从某种意义来说很有本作的特色。

总之，感谢大家读到最后，希望能在下一卷的后记再会。
