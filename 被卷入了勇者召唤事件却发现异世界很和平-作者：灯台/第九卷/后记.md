<p align="center">后记</p>

感谢您本次入手『被卷入了勇者召唤事件却发现异世界很和平』第九卷。

以前我也在后记说过，如果把到勇者祭为止的一年算作第一部，那么克罗篇是初期的高潮，菲尔篇是中期的高潮。现在，终于要真正到这时候了。

这次新登场的地球神，不只和夏罗法娜尔，还和爱丽丝有紧密的联系，可以说是相当重要的关键人物。立场上来说，她也是快人出生、长大的世界的神，今后在故事的各个关头，她也会发挥作用。

事实上，地球神这个人物，与 WEB 版相比有很大的变化，具体来说，WEB 版一开始时，她是一个『只用单词说话』的人物。随着 WEB 版进展到后半部分，只用单词的口吻逐渐变化成普通的敬语……理由？思考台词比一开始想的更加麻烦，属于作者的原因。

这一卷里，还有另一个约定，菲尔和芬芙的关系。这一点本身在前几卷中也有暗示，随着关键时候的临近，也就越来越详细了。

WEB 版中，芬芙在这时还没有登场，她和菲尔的约定关系是文库版的变更点之一。因此，菲尔篇的改稿会变多。虽然是自己做出的改变，但心里还是很慌。

说起相对于文库版的修改，在本卷还有很大的一部分，没错，就是翻开书一开始的一章，克罗姆艾娜的情节。事实上，经过 WEB 版中初期的高潮、文库版第六卷那里的克罗姆艾娜篇之后，直到六王祭附近，虽说一定程度上是没办法的事情，克罗姆艾娜的存在感都有点低，出场机会也少了。

她被说成是女主角（笑），作者也拿这点来开涮……但是，她毕竟是女主角，出了书之后加上大量的新章节，得到了出场机会。

当然，克罗姆艾娜毕竟占据了女主角的位置，所以在作者心目中是最为中意的人物，完全是在偏袒她。算下来，文库版中添加最多的，估计——不如说毫无疑问，是克罗姆艾娜有关的部分。接下来是爱西丝、伊尔奈斯这些人的新情节比较多。反倒是 WEB 版出场机会充足的爱丽丝，新情节会相对较少。嘛，关于爱丽丝，或许从下一卷开始的爱丽丝篇才是正篇吧。

总之就是这样，这一卷有很多与下一卷及更后面的故事相关的内容，您读得开心就再好不过了。

下一次是值得纪念的第十卷，让人感慨万分。这也是多亏了各位读者拿起书阅读作品才有的结果。

十分感谢您读到最后。如果您能期待后面的几卷，我会感到非常高兴。

灯台
