<p align="center">后记</p>

非常感谢您本次入手『被卷入了勇者召唤事件却发现异世界很和平』第六卷。

这一次，女主角克罗与快人的关系发生进展……在整部作品中，这一卷也可以说是一大波折了。

从名字——或者说简称，想必有人已经有所预料。克罗和希罗是关系很深的存在。这两个人对快人来说也是关键。

克罗姆艾娜是宣告快人故事开始的『起始的黑』。夏罗法娜尔是宣称自己为快人故事最终 BOSS 的『终结的白』。再说下去就是剧透，所以我无法在这里讲述，不过可以说的是，她们今后在故事里的作用也会对应于上述的概念。

要说其他部分的话，果然还是增补那部分的篇章吧。这次的增补也很多，趁这个机会，我来讲一讲与 WEB 版明确不同的地方。

首先是从第四卷就登场的伊尔奈斯。其实在 WEB 版中，尽管有稍稍提到她的存在，但她是到相当后面才会登场的。

另外就是这一卷登场的芬芙……在 WEB 版，她甚至还没有与快人相识。就我个人来讲，我考虑了很多设定，觉得是个很不错的角色，也希望能在 WEB 版让她尽早登场。然而，我找不到什么机会，所以至今仍未登场。

因此，既然有文库版的机会，我就决定早早让芬芙登场了。在下一卷之后，预定她也有一定的出场机会。第七卷可能会有她的人设图，希望大家可以期待。

另外，或许很多人都已经注意到，克罗的家人中有很多人的名字都和数字有关。

首先是德语数字 1……艾茵……接下来是菲尔 4，芬芙 5，赛克斯 6，亚哈特 8，诺茵 9。顺带一提，拉兹莉亚的名字来源于七宝的 lapis lazuli，在克罗的家人里占了 7 的位置。

一位数里面，剩下的 2 和 3 也有对应的角色，希望大家能等待这些角色的登场。不过，其实 3 对应的角色已经登场了……

类似地，尽管不是全部，但有一部分的人物命名会有章可循，比如莉莉亚家人的名字都是来源于花。这算是个小梗吧。说真的，感觉后记要没什么东西好写了，这卷的后记让我奋斗了挺久的。每一卷最头疼的部分，或许是后记吧。

那么，我再次衷心感谢各位读到这里。

下次的第一卷主角应该会是爱西丝。当然，相比 WEB 版会有很多增补，还有 WEB 版尚未登场的芬芙相关的故事。已经读过 WEB 版的各位应该也可以读得开心。

那么，我期待着能在第七卷的后记与大家再会。

灯台
