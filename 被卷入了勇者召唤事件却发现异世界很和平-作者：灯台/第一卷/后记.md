<p align="center">后记</p>

虽然应该有人读过 WEB 版了，总之初次见面，我是作者灯台。真诚感谢您本次入手『被卷入了勇者召唤事件却发现异世界很和平』。这是一本没有大风大浪的轻松悠闲的小说，希望您能阅读愉快。

刚刚把这本小说投稿到「成为小说家吧」的时候，我做梦也没想到会迎来书籍化。WEB 版的各位读者想必也很吃惊，其实我才是最吃惊的。

去年的 5 月 7 日，我投稿了本作的第一话。一段时间之内，收藏数和评分点数都很低。那时我以为书籍化简直是痴人说梦。这样的作品如今能成为书籍，让大家入手阅读，最重要的奇迹果然还是我应征了新纪元社主办的「晨星大奖」。我庆幸地赢得了大奖，让おちゃう老师这一位优秀的插画师来作画，真是感激不尽。

接下来是说，本作『被卷入了勇者召唤事件却发现异世界很和平』在书籍化时，从 WEB 版进行了修正。虽然这么说，其实并没有更改故事大致的发展，只是改换了角色的登场顺序和章节顺序而已。本作基本上以悠闲而没有压力的作风为目标，因此调整还是挺容易的。即便如此，第一次书籍化作业还是有着各种各样的辛苦。

就像标题中的「和平」一样，本作的世界是非常友善的。我的目标是「异世界中的日常作品」，因此希望您能放松肩膀愉快地享受。

这本第一卷是以女主角克罗姆艾娜为主体的故事。第二卷是否会是以 WEB 版中人气数一数二的死王爱西丝为主体呢……在协商之前我有过这样的妄想。

那么，空间所剩不多了，请让我再次致谢。

给予本作品书籍化这一光荣的机会的新纪元社。虽然我什么都是第一次，但给我周到的支持的责任编辑。尽管我只能做到粗略的说明，却仍然画出优秀插图的おちゃう老师。给我温暖的应援的 WEB 版读者。最后，还有手持这部作品的各位。用我的辞藻无法完全表达出心中感谢的心意，但请让我赠以发自内心的感谢。

真的，谢谢大家。希望大家今后也能支持本作。

最后再提供一个情报……不知道 WEB 版的人可能会觉得莫名其妙，请理解和原谅。

严肃前辈，藏在某个地方哦。有时间的话请去找找看。

灯台
