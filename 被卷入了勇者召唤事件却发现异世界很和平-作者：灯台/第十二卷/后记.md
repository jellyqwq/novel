<p align="center">后记</p>

感谢您本次入手『被卷入了勇者召唤事件却发现异世界很和平』第十二卷。

中盘的关键点，魔王菲尔相关的部分终于在这一回告一段落了。接下来的长篇，或者说是系列剧情，是之前就预告过的六王祭。

六王祭有黎格佛雷西亚三倍以上的天数，并且还是逛六王所有人的祭典，感觉会成为最长的一篇。不如说，一定会是最长的。

在 WEB 版里，六王祭的确就是最长的一段了，新登场的角色也显著更多。其实 WEB 版中，奥兹玛和伊尔奈斯是六王祭篇才首次登场的。

应该说，因为六王祭里新角色太多，所以想着先让他们一点点逐步登场，才提前了他们在书籍版的登场。

六王祭相关的就先说到这里吧，再说说本卷中的剧情。这次新登场的角色是龙王部下四大魔龙一角，弗雷贝尔·尼德贝尔特。该角色也是提前登场的角色。

她是魔界中较弱的，克罗也说过是弱小魔物的双足飞龙的特殊个体，同时也是从弱者成长为强者的存在。

在作品中没有说明，她在龙王部下中是最强的，实力远超担任部下首席的法夫纳。

要说为什么尼德贝尔特不是部下首席，是因为首席是负责统率部下的，也要综合考虑战斗力之外的因素，首席并不等同于最强的部下。

其他阵营的话，冥王阵营没有部下这个叫法，对应于其他地方首席的是艾茵，这里首席就是最强的。

幻王阵营也一样，身为首席的潘多拉是部下中最强的。

战王部下中，本卷后半稍微露了脸的阿耆尼是首席，最强的部下是奥兹玛。

界王部下首席是莉莉艾，最强部下是草华姬嘉美莉亚。这个角色之后应该会登场……虽然我还没想好是什么时候……

另外，这是非常非常后面的事情，也是 WEB 版的一点小剧透……死王部下那边是首席部下=最强部下。

总之差不多就是这样，六王干部还有很多，今后应该会慢慢登场。

嗯，我觉得后记写写这些就差不多可以结束了吧。

再次感谢大家读到最后，希望能在下一卷的后记再会。
