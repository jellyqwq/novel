艾菲娜的金黄秀发逐渐回归成奶油色。

妙舞闭幕，地狱宣告终结。

最后伫立于此的并非绝望，而是摧毁绝望的希望。

赛莲闭紧双眼。她能听见舱门被打开的声音，再来是逐渐接近的脚步声。
