# 露可 Side

被光芒笼罩的视野恢复之后，眼前是晴朗天空下的站立式餐会会场。集会似乎早已开始，红色的狼人男性满嘴都塞满了肉，金色的骸骨正一边咔咔大笑一边喝着红茶。

远处还能看到黑色的巨龙在用酒桶畅饮，旁边的独角兽正为之喝彩。

乍一看，诸多种族相互谈笑，实在是非常和平的景象。

然而，在这里的都是地下城核心。

虽然很多都是人类的姿态，但大部分都是人化而成的拟态形象。像露可和白这种从一开始就是人型的地下城核心，与那些为数不多的种族一样，都属于少数。

「那接下来……」

露可与往年一样，悄悄走向角落……今年把会场选在野外实在是令人讨厌。因为没有墙壁，也就没有了躲在角落的那种感觉。

这时，体型有人类大小的大蛇、大蛙、大蛞蝓三人（？）组走了过来。

「哎呀，这不是695号嘛，你在做什么？」

「呱呱呱，话说你还活着啊。」

「看你去年没来，DP也是0，还以为你早死翘翘了呢。」

被突然搭话，露可不由地抖了一下。

「6、650号，651号，652号……好、好久不见。」

「啊！？要在称呼前面加第，695号！」

「噫！？」

砰！650号核心大蛇用尾巴敲击地面，发出巨大的声响，露可也反射性地绷紧了身躯。

顺带一提，在地下城核心之间，加『第』是敬称。对露可来说，绝对不想以此来称呼它们。

「什么啊，都是600位的序列，这都无所谓吧。」

「啊，听不见。呱呱呱呱……排名最低的695号到底在说些什么呢？」

像是要阻断露可的退路一样，651号核心大蛙从右后方绕到了前面。

「是啊是啊，太狂妄了。」

「噫……」

652号核心大蛞蝓将手（？）放到了露可的肩上。这正可谓是三位一体，三难困境的阵型……露可立刻就起了鸡皮疙瘩。

大蛇靠近露可的脸，不时伸出分叉的舌头。

「嘶嘶……呵呵呵，稍稍去那边一下吧？」

「不、不要，放开我。」

「放心放心，不会死的，呱呱呱！」

「好了，来一起黏黏糊糊吧。」

那边正是露可本想去的角落，但是，以自己的意志过去和被这三人带过去，这两者之间有很大的区别。况且，露可之所以偷偷摸摸的，就是为了不被这些家伙发现。

没有人去在乎露可被强行带走……终归就是最低级间的恶作剧或是扯后腿而已。

就像是糖块掉到地上引来蚂蚁也没人在意一样，对这里大多数地下城核心来说，糖块、蚂蚁、露可以及这三人都毫无价值。

「喂，能打扰一下吗？」

这时，红色的沙罗曼蛇出现了。

112号地下城核心，伊哲。

整整一座山基本都是其领域的51层地下城，他便是那个『火焰窟』的地下城核心。

和主人兼妻子的红龙一起充分享受着悠闲的生活，虽然称不上是顶级，但也算的上是踏实而稳定的中坚级地下城。

而且是100位的序列，在资历上也很老。

「第、第112号地下城核心大人！？找我们有什么事吗？」

「啊？啊——……呃，你们都无所谓。我找的是她，695号，过来。」

「诶？请问，您指的是这家伙吗？」

「啊？有意见吗？你是几号？」

「我是6、650号！那、那个，第112号大人，我是您的粉丝！」

由于蛇没法握手，所以伸出了尾巴。

然而伊哲无视了它，转向了露可。

「哦。好了，我们走，695号。」

「诶？呃……嗯。」

在伊哲的带领下，露可从三人的包围中解放了。

650号核心保持着伸出尾巴的姿势，哑然的目送着露可离去。

……露可用【净化】洗去蛞蝓的黏液，尝试着向沉默着走在前面的伊哲搭话。

「那、那个，第112号……大人？」

「啊！？白痴，不管是“第”还是“大人”都不需要加。话说，你干嘛任凭那些家伙欺负？」

伊哲回过头，那训人的样子依旧像是要吃人一样。

资历较老的伊哲多少有些名气。因此，和露可刚才被带走那时不同，稍稍引来了一些人的注目。

「可是，像我这种底层地下城……被三个人围攻，不可能去反抗的吧。」

「哈？哦，哦。好吧好吧。」

伊哲用他那特有的火尾巴来回戳着露可。

虽然知道他明显没有用力，但露可还是感到很痛。而且还有点热。毕竟刚得到他的帮助，所以也不好说什么。

「呵呵呵，别太妄自菲薄了，你可是连我都赢过！」

「……那全多亏了桂马，不是我的力量。」

「哈哈哈，确实，虽说诚实是好事……但你赢了我也是事实。所谓核心的实力，主人也是包括在内的哦？」

“也就是说，我和妻子的爱情力量也包含在内”，伊哲笑着如此这般的说道。

也不知是听到了两人之间的对话——还是伊哲故意大声说话的缘故——周围不时传来「……赢了？」「赢112号……？」「695号……不是最低的那个吗？」的声音。

其中最刺耳的便是「到底用了什么卑鄙的手段？」，实际也确实用了卑鄙的手段，这边也无可反驳。说到底，那场地下城对战本就应用了特殊的规则。

「……可桂马说那场对战是平手哦？」

「那下次试试变换一下攻守吧？」

「不好意思，容我婉拒……那我在这里和你分开吧，谢谢你帮我，112号。」

露可本想离开，但被伊哲叫住了。

「等等，『父亲』就快要致辞了。在那之前一起吃个饭呗？这个肉可是相当美味的。」

「……就算地下城核心不吃东西也能活下去。」

「这个桌子上没人投毒或是把垃圾混进去，所以放心吧。难道你不相信我？我可是恩人哦？」

「……唔，那就稍稍吃一点。」

露可怯生生地咬了一口伊哲递出的肉。

咬上去的瞬间，肉汁立刻在嘴里扩散，舌头被浓郁的肉香包裹。

……虽然不清楚是什么肉，但相当好吃。露可感觉妮库应该会喜欢这种味道。

「噢，嘴边变得一塌糊涂了哦。来，给你个【净化】。」

「唔咕，嗯。谢谢。」

「哈哈哈！不客气！」

就这样，露可吃饱喝足之后没过多久，声音在会场上响起。

『嗨，我可爱的孩子们。大家都别来无恙吗？』

抬头望去，浮在空中的巨大荧幕上映出了一个男性的身影。

地下城核心们一同看向了荧幕里的男人。露可自然也包括在内。

「父亲大人……时隔两年，终于能再度拜见您的尊容了。」

除了被称作『父亲』『父亲大人』以外，还有人将其称为『王』『神』。那是个浅黑皮肤黑发金眼的男人，身穿着藏青色的法衣。虽然露可刚才说的是尊容，但他的眼部带了面具，面具没有覆盖住的嘴角则带着淡淡的微笑。

在场的地下城核心都不知道他的具体身份。

但至少能够确定，这里的所有地下城核心都是他创造的。这一点是无可动摇的事实。

他正是『第0号地下城核心』，也有的地下城核心认为所有的核心都等于是他的『伪装核心』，而且这种观点还得到了一定的支持。

多数地下城核心之所以要化作人型，那也是因为『父亲』是人类的形态——总之，他是以人类的姿态显现在大家的面前。

因此，从一开始就是人型的白也成为了大家嫉妒的对象……白故意将露可的地下城名次降低，其中的意图也包含了露可就能借此免遭他人嫉妒的层面。

『哎呀，是不是少了一些？真令人悲伤。差不多该补充新的孩子进来了。你觉得呢，1号？』

「是，如果父亲是那么想的话，那就应该那么去做。」

骑士外形的1号核心对『父亲』的提问进行了回答。除去『父亲』以外，他是这个地下城聚会中最强、资历最老、最具影响力的存在。一位数的号码并不是装饰。

『嗯—，好无趣的回答。有其他孩子持不同的意见吗？』

不过，『父亲』似乎对1号核心的回答不是很满意，于是又开始征询其他核心的意见——但没有人提出自己的意见。确切来说是不敢提出意见。

敢在这时候提意见就代表着和1号核心唱反调。更何况，向『父亲』提意见，实在是太过惶恐了。

「我能提一下意见吗，父亲大人？」

一名核心在众人当中举起了手。

是哪个不要命的——所有人的视线都汇聚到了一处。

『说吧，89号，你能给我其他意见吗？』

『父亲』的声音里带着些许喜悦。89号核心……白的脸上展现出了微笑。

「600号序列活下来的还有七成左右。增加新的弟妹可能还为时尚早。等减少一些之后再去创造应该会更好吧。」

听到她的话，一名地下城核心轰地向空中喷出了黑色的火球。

「你这『叛徒』还有脸说！」

那雄浑的声音几乎让大地都颤动起来。

5号核心黑龙强行闯进了白与『父亲』之间的对话。但『父亲』温和地对5号核心进行了责备。

『5号。我现在正要听89号的话。你有什么话，接下来再听。』

「是、是我失礼了，父亲！可是！」

『接下来再听，我刚才是这么说的吧？5号你是聪明的孩子，会乖乖等着的，对吧？』

「是……」

被叮嘱要乖乖等着，若是再纠缠不清的话就等于是在否定『父亲』所说的话，5号只能退到一边。

接着，『父亲』再次向白发问。

『那么，89号认为要减少到何种程度之后再增加新的孩子呢？』

「差不多等600号序列的残存数不到一半的时候吧。之前也都是在那时候增加的弟妹。」

『原来如此。具体且易懂的意见。谢谢，我会作为参考。』

「能得到您的褒奖，深感荣幸。」

白一边说着，一边恭敬地低下了头。

『那么，接下来听一下5号的意见。说吧。』

「是！父亲，那个89号是『叛徒』，专门猎杀自己的同胞。死掉的地下城核心大多数都是89号干的好事。正因为如此，她才能详细掌握了地下城核心的残存数……我认为，89号的意见根本就没有值得听的价值！」

『嗯。89号，是那样吗？』

「是的，我为了能活下去，确实排除了其他的地下城核心。父亲大人，这有什么问题吗？」

『没有。为了活下去，牺牲其他人——这是非常正确的态度。嗯？好像去年也说过差不多的话来着。』

「每年都是这样，父亲大人。」

「是吗。在场的都是我可爱的孩子。要注意分寸哦。」

「当然，谨记在心。」

白微微笑道。

没错，白被训斥为『叛徒』是每次都会发生的事。『父亲』让她注意分寸也是一如既往的事。

『于是，5号你怎么看？对于增加新的孩子，有什么意见吗？』

「那、那个……我认为，只要父亲高兴，那就是最好的。」

『嗯，谢谢。顺便也问问其他孩子的意见吧……』

『父亲』略显无聊的呢喃着，环顾寻找着下一个提问对象。

『噢』

他的目光定在了露可所在的方向。

露可不由地颤抖了一下……难道要被点名了？会被点到？她不安地游移着视线。

『112号，你呢，你有什么意见吗？』

原来是那边。露可安心地吐了一口气。

「啊……呃，第5号说的，只要父亲高兴，那就是最好的……我也同感。再其次的话，如果有增加新的孩子，我会非常期待。就算现在没有，也会耐心等待的。」

『你很期待啊。那真是再好不过。』

强制自己使用说不习惯的敬语，伊哲表现的有些滑稽，『父亲』则是微笑着听取了他的意见。5号核心向博取『父亲』欢心的伊哲投以了赞扬的眼神。

『那么，顺便问问旁边的695号。你怎么认为的？』

「……诶！？」

突然被『父亲』搭话，露可不由地绷紧了身躯。

……不要搞突然袭击啊。露可一边在心里想着，一边慢慢说道。

「那、那个……我、我是600号序列里最后面的……如果有后辈的话，我也会高兴……但是，如果被赶超的话，则会有很复杂的心情。」

『哈哈哈，复杂的心情。这很有你的风格，695号。』

露可的脸红了起来。既然说是「很有你的风格」，那就等于是『父亲』对露可非常关注，以至于留下了「露可的风格」这样的印象。

果然DP排行榜最低同样会很引人注目吗？

另一边的白则用和颜悦色的笑容面向了刚刚做出发言的露可。

露可轻舒了一口气。她自己也很清楚，刚才的情绪非常紧张。

之后『父亲』又问了几个核心的意见。但他们的意见基本都是「只要『父亲』高兴就好」这种和1号核心差不多的内容。

——我也那么说的话，说不定就不会引人注目了。

露可事到如今才刚想到这一点。

『嗯，那今年就算了，明年再考虑。』

『父亲』说着，拍了拍手。

场面里的气氛随即缓和了下来。

『大家都听好。如果想对我提出要求或者疑问以及意见的，需要在这场宴会的期间说出来。若不是有这样的机会，我基本无法干涉你们的事情。』

『父亲』没有被面具掩盖的嘴角展露出了笑容。

有一个地下城核心——人化为壮年男性的核心发出了声音。

「噢噢，我等的神明啊！能否聆听一下我的愿望。」

『真是够快的。何事，380号，不管什么事都可以说来听听。不过，如果能称呼我为父亲，我会很高兴。』

「实在不敢当，把微不足道的我当作儿子……我一直都是神明的仆人。」

『啊——嗯。然后，你的愿望是？』

380号核心跪在地上，说出了愿望。

「是！请赐予代表神明威光的奇迹！请您将死者复活！」

『哎呀？让喜欢的孩子复活，记得好像不能做来着？』

「我请求复活的是我的哥哥379号核心！去年冬天，由于遭到89号卑劣的魔手，成为了邪神尖兵的饵食……」

『啊……原来是379号。你们是双胞胎呢。嗯，很抱歉，那是不行的。』

「请问这是为什么！？如果是神的力量那应该！！」

不是「做不到」，而是「不行」。两者之间有很大的区别……如果是『父亲』的力量，就算是被破坏而死掉的核心也是可以复活的。

然而，『父亲』像是教导380号核心一样，无情地宣告道。

『因为有那样的规定。』

那句话代表着，即便是『父亲』，也受到了某种规则的约束。

『父亲』拍了一下手，像是想到了个好主意。

『对了！就让你能够制作和379号相同外表的怪物吧。那孩子也和你一样能人型化来着？把那个也重现出来好了。虽然要用一些DP，但那可是能以假乱真的怪物哦……好了，已经完成了。这样一来380号就不会寂寞了吧？回去想做多少都可以。』

380号在一瞬间露出惊讶的表情，随即像是叩拜一样低下了头。

「那……不，神啊，感谢您的慈悲。」

『嗯，你能开心就好。』

380号核心的头伏在地上，身躯不停颤抖着。

他真的是因感动而高兴吗。露可无法去理解。

『那么，感觉大家早就翘首以盼了，那就来发表DP排名。』

听到『父亲』的话，现场的气氛再次紧张起来。

『好的，大家注意了。首先公布前十位！不过，这几年的排名基本没有变动，直接就放出来吧。』

空中出现的巨大荧幕上以榜单的形式一口气公布了第一名到第十名。

和露可知晓的去年结果相比，排名没有变化。

第一到第三名分别是1号、2号、3号核心。第四名是7号核心，第五名是9号核心，第六名是6号核心，第七名是89号核心白，第八名是5号核心，第九名是10号核心，第十名是8号核心。

噶哦哦哦！5号核心很不甘心地发出咆哮，向空中喷出了黑炎龙息。

前十名基本都由单位数号码的核心占据。露可认为能在那里面占据一席之地的白果然非常厉害。

另外，虽然DP的持有量也会一同显示——但露可并不认识兆以上的单位，因此她也知道那是件非常了不得的事，毕竟连看都看不懂。

『嗯，果然1号、2号、3号很优秀。我很欣慰。就以这个态势继续去收集DP吧。』

「能得到您的夸奖，欣喜之至，父亲。」

1号核心单膝跪地，像骑士一样恭敬行礼。2号核心和3号核心也跟着效仿，纷纷低下了头。2号核心和3号核心分别是银色、铜色骑士的外表，很是威风凛凛。在被称为最初序列的1~99号当中，这三名也是『父亲』精心创造出来的，在某种意义上他们获得前三位也是理所当然的。

不过，他们都收集了几乎无法认知的大量DP，却还被嘱咐要「再收集更多」，露可再度认识到彼此之间有次元上的差距。

『7号和9号也很努力了。今后最好能超过3号啊。』

「是，父亲大人。」

「虽是绵薄之力，但我会尽力而为。」

7号、9号核心分别是具备神秘气质的绿发和蓝发女性。似乎还被称为『山神』、『海神』。毕竟白也被称为『白之女神』，露可觉得应该都是差不多的东西。

『6号、89号、5号一直都在照顾排名靠后的那些孩子。提高整体水平也是很重要的，谢谢你们。』

「您言重了。」

「能让父亲大人高兴，那真是太好了。」

「父亲，我5号核心虽然还不成熟，但能为您派上用场，感到非常高兴。」

6号核心。身穿阴森的铠甲，庄重而威严的中老年男性。通称是『大魔王』，其派系被称为魔王派或是魔王军。
