种族：兽人（推测）

知能：高（推测）。推测程度和吸血鬼相同。

外观：有着银色兽毛的兽人。像是狐狸进化成了两足直立型一样的身姿。身高和人类相同。

魔素：中等程度（推测）

自报名称的怪异，自“大敌”艾尔莎丽夏以来这是第二体。

威胁：低（相较于其他“大敌”）
