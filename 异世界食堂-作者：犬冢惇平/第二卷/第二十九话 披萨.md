<p align="center">第二十九话 披萨</p>

在王都里没有人不认识西里乌斯。艾尔费德这位艾尔费德商会的下任当家，他继承了现为艾尔费德商会当家的父亲，以及曾是拥有相当家世的贵族家千金的母亲的血，西里乌斯既是大商会的继承人，同时也是真正的贵公子。

他出生时，艾尔费德商会已经获得王城御用商人的地位。而且在西里乌斯懂事前，他的叔父们就已经被派去帝都与公都建立分店，让商会进一步变得更加繁荣。

西里乌斯的祖父――艾尔费德商会的中兴之祖汤玛斯在约半年前将某个秘密托付给这样的他，那个秘密就是现在的艾尔费德商会的基石，汤玛斯长年独自隐藏……已经藏了超过二十年的某间店，汤玛斯将通往那个场所的门托付给西里乌斯。

西里乌斯一开始非常惊讶。西里乌斯是以贩卖食品为主的艾尔费德商会的大少爷，但那间店却有许多连他都没吃过的料理，而且还非常美味，他当时津津有味地享用了那里的料理。

然后他发现一件事，如果「继续」窃取这间店的「味道」，一定能让商会变得更加繁荣，但这么想的人不只自己。

将艾尔费德商会各分店搜集来的情报，以及西里乌斯亲眼观察「异世界食堂」后搜集到的情报整合在一起后，可以看出许多事情。

例如在东大陆拥有最多的信徒，信奉光之神的教团中，有一位虽然很会烤面包但没什么当祭司的才能、结果不知为何被调到了某间能干女祭司云集的尼僧院的见习祭司，以及会带年轻的女祭司弟子们去那间店，喜爱磅蛋糕的光之神高祭司。

例如之前为了追求新的交流与交易而来到帝国的西大陆沙之国的使者。所献上的一种类似用魔法冷却过的冰卡发的饮料，以及一对喜欢那个叫漂浮冰咖啡的饮料，近似沙之国贵人的兄妹。

例如矮人的城镇新发明的又烈又好喝，连艾尔费德商会都想知道作法的新型烈酒，以及总是吵吵闹闹地大口喝酒的矮人二人组。

没错，这世界还有其他人抱著和汤玛斯相同的想法，而且就某方面来说，西里乌斯比谁都清楚只要肯花时间并持续努力，就能让那样的想法开花结果……所以他才不能眼睁睁地待在一旁观看。

于是西里乌斯开始行动。他要做过去在艾尔费德商会发展成大商会前，汤玛斯因为害怕秘密泄漏而无法做的事情。

如今即使是在王都，艾尔费德商会也算是个特别突出又实力深厚的大商会，外加西里乌斯还有一个「符合条件」的好友，所以他才能采取这种手段……

「少爷，你也差不多该告诉我有什么事了吧。」

艾尔费德商会专属的年轻厨师，乔纳森向西里乌斯问道。

――我有重要的事要告诉你，不好意思，乔纳森，你可以先什么都别问直按跟我走吗？

说完这些话后，西里乌斯便带乔纳森来到艾尔费德商会的某间旧仓库。

因为附近就几间气派的大仓库，所以这间仓库现在几乎没在使用，乔纳森完全猜不出来这里究竟有什么。

「……嗯。时间差不多了！」

另一方面，西里乌斯在走进仓库确认周围没有其他人后，才对乔纳森说道：

「我今天想跟你一起吃个饭。」

「……吃饭？和我吗？」

西里乌斯的话让乔纳森困惑地问道，乔纳森的父亲平常会与艾尔费德商会一同研究新商品，是名非常被重视的厨师，在这样的背景下，他与年龄相近的商会大少爷西里乌斯的关系从以前就非常亲密，在跟父亲一样成为艾尔费德商会的厨师之前，乔纳森也常陪西里乌斯在王都内到处跑。

不过即使两人过去曾经一起去摊贩或店里吃饭，西里乌斯也从未如此正式地邀请乔纳森一起用餐。而且他们最后抵达的地方是艾尔费德商会的旧仓库，这里看起来实在和用餐沾不上边。

「嗯。至少除了乔纳森以外，我不打算带任何人去那里。」

「……那里？」

西里乌斯告诉困惑的乔纳森真相，告诉他那个在艾尔费德商会里只有自己和祖父知道的秘密

「没错，就我所知，那是一间提供的料理和这个世界不同，但真的非常美味的店……」

在艾尔费德商会的旧仓库内，出现了一扇绘有猫咪图案的黑门，让艾尔费德商会发展成现在的大商会的契机，就在那扇门的对面。

「异世界食堂。」

那一天，乔纳森·威兹帕克成了第三个知道这扇门存在的人。

伴随著来客铃的铃声，穿过门后突然从阴暗的仓库来到某个明亮场所的乔纳森，首先吓了一跳。

「少爷。这里真的是异世界吗……唔哇？」

东张西望地环视店内……在与吃著堆积如山的蛋料理的蜥蜴人对上视线后，乔纳森忍不住发出惨叫。

「放心吧，这里的客人就算是魔物也不曾袭击人类。」

乔纳森的反应让西里乌斯回想起自己刚来这里时的状况，并因此露出苦笑，接著他熟练地随便找了个位子坐下。

「欢迎光临。」

没过多久，店长便走出来将菜单轻轻放到桌上，最近变得常来这里的西里乌斯除了最喜欢的拿坡里义大利面以外，通常会再加点一样不同的料理。因为知道这点，所以店长立刻拿了菜单，柠檬水和湿毛巾过来。

「啊，谢谢。」

另一方面，西里乌斯虽然一如往常地收下菜单，但没看就直接放到旁边。

「不过我今天已经决定好要点什么了。」

说完后，西里乌斯讲出那道料理的名字。

「请给我两片洋葱和培根的披萨，分别是我和他的份。」

这就是他今天想让乔纳森吃的料理名称。

「好的。」

店长点头回答，然后就直接返回店内后方。

「……那么，那个洋葱培根披萨到底是什么料理？」

目送店长离开后，乔纳森向开始用店长送来的湿毛巾擦手的西里乌斯问道，乔纳森从未听过那种料理，所以坦白讲他完全无法想像会出现什么样的东西。

「嗯，如果是乔纳森，应该吃过就知道了。」

与之相对，西里乌斯以宛如在策划什么恶作剧般的纯真笑容冷淡回应。

(唉，既然是异世界的料理，那就算听了也没用。)

乔纳森模仿西里乌斯用温毛巾擦手并转换想法。西里乌斯少爷说这里是异世界……这应该是真的。

连在大陆最大的城市王都也不曾看过的客人，店里那些远远超出乔纳森常识的设备，以及明显不会出现在王都的客人们。原来如此，创这里是异世界反而比较能接受。

(……喔。这个水好像加了什么柑橘类的果汁。)

边思考边随□喝下的透明杯内的冰水带著些微清爽的风味，比一般的水还好入口，这让乔纳森理解一件事，能不经意地做出这种贴心举动的店长，技巧应该远比一般的厨师高超。

(不对，既然是少爷常来的店，这也是理所当然。)

身为在王都贩卖食材和料理材料的商会继承人，西里乌斯平常吃的料理都是由厨艺在王都也算是顶尖的艾尔费德商会的精锐厨师烹调，既然能得这位舌头可能比一般王族还要挑剔的青梅竹马欢心，这里的厨师一定厨艺高超，料理也非常美味。

察觉到这点后，乔纳森和西里乌斯一起等待那道叫披萨的料理。

「久等了！这是洋葱培根披萨！」

料理没过多久就来了，送来的人不是店长，而是一名年轻的魔族少女。

(……这是，加了起士的面包吗？ )

乔纳森仔细端详那个刚烤好还在滋滋作响的料理，装在盘子上的是被切成六块，看起来像薄面包的东西，面包上面加了捣碎某种红色的东西后做成的酱汁与被切得细细的欧拉尼耶，另外还放了起士、熏肉和被切成丝的蔬菜。

「我们开动吧，这个趁热吃特别好吃。」

说完后，西里乌斯立刻开始吃披萨。他没用刀叉，直接用手抓。看来这种料理就是要这样吃。

(到底会是什么味道……)

另―方面，乔纳森对这种未知的料理感到困惑。既然西里乌斯说吃了就知道。那也只能吃吃看了。抱著这样的想法，乔纳森从六块里挑了一块来吃。

看来这料理和外表看起来一样????是用烤的。将融化的起士拉断后，乔纳森对热到足以烫伤手的料理稍微皱起眉头。

(嗯，味道还不坏……嗯？这是……)

披萨的料理散发出起士和培根的油脂在烤过后混合在一起的香味、因为从中感觉到一股熟悉的味道，让乔纳森在心里困惑了一下。

(……算了。吃过就知道了。)

即使对那股不协调的味道感到困惑，乔纳森还是咬了一口披萨。

(嗯？果然不出所料！ )

融化的起士那带有少许酸味的乳香味，以及经过仔细烧烤后去除了多余的油脂、优质的熏肉味。将这些味道进一步提升，直接拿去烤熟的欧拉尼耶轻微的辣味，以及放在上面的绿色蔬菜独特的苦味。

充当这些配料基底的薄面包，被烤得外硬内软，推测只用优质小麦、盐和水制作出来简单面包清淡的味道，支撑著各种配料强烈的味道。

不过最令乔纳森惊讶的并不是这些。

「少爷？这是……马尔梅特吗？」

没错，夹在配料和基底面包之间的东西，同时包含了酸味与惊人的美味，艾尔费德商会在发现某个边境小国有种植后，费尽千辛万苦才成功在王国培育的蔬菜，马尔梅特。

这个料理是用除了艾尔费德商会以外，几乎没人知道的马尔梅特来当酱汁的基底。

「嗯。虽然这在异世界似乎被称作番茄……不过没错，这是一道以马尔梅特为主的料理。」

像是觉得乔纳森果然没有辜负自己的期待般，西里乌斯满意地点头，没错，就是因为拥有能看穿这点的舌头，乔纳森才能自称是艾尔费德商的厨师，带他来果然是个正确的决定。

「我晚点再说明我的打算……先继续吃吧。」

西里乌斯看向马上就想继续「研究」的好友如此说道。

「喔，喔 我知道了。」

乔纳森点头，将手伸向剩下的披萨。

(这是用热水烫过后剥皮再捣碎……不对，看来相当费工夫。 )

惊讶归惊讶，乔纳森毕竟是已经踏入一流世界的厨师，他每吃一口眼前的料理，就会逐步进行分析。这个酱料基本上和乔纳森协助开发的肉酱非常接近，是一种对剥过皮的马尔梅特进行各种调和制造出来的酸味强烈的酱汁，就算是乔纳森，如果想做出相同或是更胜一筹的味道，也需要进行相当的研究。

(而且没想到面包居然还能这样用……)

用面包代替盘子的想法并不新奇，但从来没人想到「在烤之前」就把配料放上去

一起放进窑里烘烤。不过乔纳森能理解这么做的意图，藉由这样的作法，能让面包与配料变得像长相厮守的夫妻般契合，不论是充满酱汁与配料的滋味，又薄又坚硬的中央部分、还是较为厚实并带有强烈小麦香味的边缘部分，吃起来都很美味，对披萨而言，面包已经不是单纯的附加物，而是一样货真价实的配料。

(只要在商会研究，原来如此。)

吃到一半时，乔纳森发现西里乌斯的意图。西里乌斯的目的是……

「少爷，不好意思，在仔细品尝味道之前就吃完了可以再点一份吗？」

「那当然，我就是为此才带你来的呃，阿蕾塔小姐，请再给他一份培根披萨。然后我要拿坡里义大利面。」

西乌斯对察觉自己意图的乔纳森露出微笑，同时向正好经过的服务生加点，他帮乔纳森点了跟刚才一样的东西，帮自己点了比祖父推荐的肉酱义大刊面还要喜欢的料理。

「好的！谢谢惠顾！」

服务生精神抖擞地回应。两人的用餐……不对，研究还会继续持续下去。

「不过那个叫披萨的料理，完成度真的高得可怕。」

从异世界回到艾尔费德的仓库后，乔纳森感叹地说道，最后乔纳森总共吃了三片培根披萨，他一直吃到肚子觉得痛苦，舌头已经确实记下那个味道，随时都能研究如何制作的程度。

「嗯，只要改变配料，就能做出各种变化……最棒的一点是如果没有马尔梅特就无法成立。」

西里乌斯点头回应乔纳森。虽然使用马尔梅特做成的各种酱料确实很美味，但在王都还没多少人知道，只有艾尔费德商会能稳定地供货……换句话说，只要让王国人民知道马尔梅特，让他们像祖父和自己一样著迷，就能让艾尔费德商会获得更多的利益。

所以西里乌斯才会选择披萨作为第一个让乔纳森品尝的料理。

虽然令人耳目一新的异世界作法与丰富的应用方法也很重要，但最重要的还是那个使用马尔梅特的披萨酱汁。只要完成那倜，就能取代骑士酱，成为艾尔费德商会下一代的主力商品。西里乌斯认为披萨有这样的潜力。

「话说少爷，关于那间店……」

乔纳森以充满期待的眼神看向西里乌斯。他明白如果想使用那扇门，就需要获得眼前这位朋友的许可。

「嗯，有机会我会再带你去。不过那里原本就只能七天去一次。」

西里乌斯按照预定计画，苦笑地对好友说道。被锻炼成艾尔费德商会的精锐，年轻又有才能，最重要的是不必担心会背叛的厨师。对因为急速成长而经常树敌的艾尔费德商会来创，乔纳森是位难得的人才。告诉他这个秘密是个正确的决定。

「谢谢。」

乔纳森毫不隐藏自己的笑容低头道谢。

(那里应该还有许多好吃的东西……虽然对店家不好意思，但就让我把那些偷学过来吧。)

乔纳森一面回想眼前的朋友刚才吃的那道使用了马尔梅特酱汁的面料理，一面暗自下定决心。
