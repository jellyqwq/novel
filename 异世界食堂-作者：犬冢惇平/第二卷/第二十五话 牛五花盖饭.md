<p align="center">第二十五话 牛五花盖饭</p>

现在离太阳在南方的天空闪耀还有一段时间，在海国宫殿的最深处，菲丽正在自己的房间准备出门。她看准平常负责替自己换衣服的女官外出的时间，开始帮自己换衣服。

她穿的是没有施加任何用来保护玉体的阴阳师咒术、宫殿里最便宜的女官服。连薰衣服用的香，都是用比依法只有菲丽的族人能使用的最高级薰香还要差好几个等级的东西。

菲丽的母亲在嫁到这个国家前曾是东方大国「王国」的公主，菲丽的眼睛也遗传了母亲的颜色，不过无论是与其眼睛颜色相同的青玉发簪，还是她平常爱用的颜色与自身头发相同的黑珍珠耳饰都太过显眼，所以她将这些宝石类饰品全部摘下，就连妆也没化。她接下来要去的地方不需要这些东西，更重要的是这样会破坏菲丽想做的「变装」

(嗯，大概就这样吧。)

换好衣服后，作为最后的收尾，她拿出专门侍奉宫廷的工匠用银板打造出来的镜子确认全身的状况，虽然继承了异国出生的母亲的血、芳龄十四岁就被宫廷内的小丑与诗人誉为「倾国」的独特美貌实在无法隐藏，但除此之外都伪装得很完美。

第

(现在的吾怎么看都是个平民女孩。一定谁也无法察觉吾的真实身分。)

菲丽陶醉于自己完美的变装，并相信自己现在的打扮就是从未见过的「平民女孩」的样子。

(那么出发吧。)

大致做好准备后，菲丽小心翼翼地躲过啰唆女官的视线外出。她前往位于宫殿中庭的花园。四季都曾盛开不同花朵的那里，就是通往菲丽目的地的「通道」。

除了冬季以外总是会开满各种花朵的花园，充满芬芳的香味。菲丽走进位于花园角落的林子里面。

(……那个狐狸男今天果然真的要参加酒宴。 )

一踏入林中，菲丽就发现这里没像平常那样施加会让人迷路的咒术并放轻脚步。

菲丽从负责照顾她，平常与她交情良好又爱听宫中八卦的女官那里听来的消息没错。

那位魔法技巧高超但感觉难以信任、让人联想到狐狸的宫廷阴阳师，今天离开宫殿去参加负责外交的宰相主办的酒宴了。在为了公务或宴会离开宫殿的期间，那位海国最优秀的阴阳师会解除平常设在这附近、不让任何人靠近「那里」的迷路咒术。

……那种情况和「DOYOU日」重叠在一起的日子好几个月才会遇到一次，菲丽就是在期盼那样的日子。

然后菲丽终于见到了她的目标。

( 三个月没见到了。 )

绘有猫咪图案的黑门，那里正是菲丽不惜像这样变装成平民，也要瞒著那名阴阳师前往的场所，一想起好久没吃的「那道料理」，就让菲丽忍不住咽了一下口水，她握紧门把……打开门。

伴随著来客铃的铃声，菲丽穿过门抵达了那里。

「哎呀，欢迎光临，好久不见了呢。」

「嗯。有劳你出来迎接了……照旧给吾那个就好！」

说完后，菲丽对店长轻轻一笑。有一天，菲丽发现有个旅小人(半身人)擅闯宫廷庭园，作为放过他的代价，那位瘦小到让人联想到老鼠、自称诗人的旅小人带她来到了这里，自从吃过了那个，菲丽就决定以后都要点那道料理。

「好的，请稍等一下！」

已经习惯的店长在回答完后，便走回店内后方。没错，这里是异世界食堂。令菲丽深爱不已。贩卖在海国绝对吃不到的「那道料理」的店。

(呵呵……这里还是一样热闹。 )

随便找个位子坐下等待料理的期间，很少有机会来到热闹餐厅的菲丽静静地观察周围。异世界食堂有各式各样的客人。不论种族或出身都完全不同的客人们各自吃著自己的料理。其中也有几名出身和菲丽一样高贵的客人。

例如某位正在吃一种又黄又软的料理的女子，那人拥有带蓝色的银发与浅蓝色眼睛、身为半森贤人(半精灵)证明的尖耳，以及宛如作工精致的人偶般的美貌，菲丽小时候曾见过那个人，她是大贤者亚尔托利斯的弟子魔女公主。

例如某对正在享用上面加了一球白色冰淇淋的黑色卡发，以及上面加了霜淇淋的漂亮又清澈的绿色饮料的兄妹，他们应该和父亲的其中一位侧室一样是来自沙之国，因为穿著华丽的两人都拥有褐色的肌肤和深邃的五官，以及黑珍珠色的眼睛和丝绸般柔顺的黑发。

此外那两名沙之国的贵人似乎正在关注一名少女，那位拥有菲丽从来没见过的淡金色头发与透明白皙的肌肤的美丽少女，正在吃一种宛如工艺品般华丽、装在玻璃杯上的点心。

(呵呵……大家都太不识货了。 )

他们吃的每―样东西，都远远比不上菲丽所知的「这间店最美味的料理」。对此偷伦产生优越感的菲丽静静地等待那道料理。然后――

「久等了！我送料理来了……呃，应该没送错……吧？」

「嗯，吾点的就是这个。」

店长新雇用的魔族服务生不知为何困惑地进行确认，菲丽点头加以肯定。服务生端来的碗里，确实装了令菲丽心爱不已的那道料理，菲丽远远就闻到那股香味，并不自觉地咽了一下口水。

「那慢用。」

或许是接受了菲丽的说明，魔族服务生轻轻将用味噌调味的汤、连同冰块一起倒进玻璃杯内的饭后茶，以及那道料理放到菲丽面前，那道料理只用到一样餐具，那就是一个大碗，在绘有白色与蓝色横条纹的瓷碗内，装著白米和颜色鲜艳的蔬菜……以及经过确实调味的烤牛肉。

「啊啊……这就是让吾不晓得期盼了多久的，异世界的牛五花盖饭。」

隔了三个月才见到的那道料理，让菲丽忍不住发出惊呼，距离第一次被旅小人带来异世界食堂并吃到那道料理，已经过了两年。对菲丽来说，最棒的料理就是这道「牛五花盖饭」。

「请慢用，餐后我会再送甜点过来！」

目送急忙去服务其他客人的魔族服务生离开后，菲丽终于开始准备吃牛五花盖饭。

(首先把碗端起来……享受外观和香味。 )

菲丽端起碗，遵循那天即使是在涉世未深的菲丽眼里也显得可疑，宛如老鼠般的旅小人笑著告诉她的「最能享受牛五花盖饭的异世界吃法」。

菲丽用手掌感受为了避免料理变冷而事先温热过的碗传来的热度，用瀸细的手臂感受碗的重量。大碗装的料理对菲丽纤细的手臂来说有点重，被拿到眼前的「牛五花盖饭」不断诱惑著她。

碗里的配料多到让人看不见底下的白饭，外观呈鲜艳橙色并带有些许甜味的卡琉特、煮熟后仍是深绿色的凤草、透明又白皙的细长森贤人(精灵)豆芽，以及撒了类似白色种子的东西，被调味的酱汁染成褐色的肉。

菲丽将料理拿到自己面前，轻轻闻了一下。被吸进形状姣好的鼻子内的，是刚煮好的白米甘甜的香味，刚烤好的肉的香味，以及调味用酱汁又甜又咸的刺激香味，这三种味道混合在一起产生的香味，让她觉得喉咙深处……觉得自己的胃似乎轻轻响了一下。

(啊啊……吾再也忍不住了！ )

她遵循从海国传统礼仪的角度来看应该会显得相当低俗的「异世界吃法」，开始享用牛五花盖饭，首先夹起吸满汤汁的肉片，像小鸟啄食树木的果实般稍微含在嘴里，再用牙齿咬断。

(啊啊，这个味道谊！吾不晓得在梦里梦到了几次！)

在舌头上扩散开来的，是柔软肉片包含的肉汁，以及口感细微的油脂的味道，海国宫殿端出的肉和鱼，都会先经过长时间的炖煮或烧烤以彻底排除内在的「污秽」，所以根本无法释放出现在允满嘴里的肉味，而支撑著这股肉味，将味道提升到艺术境界的，就是用来替肉调味的酱汁。

那个酱汁带有水果的甘甜、少许辛香料的辣味，菲丽从未听闻过的某种类似鱼露但确实不同的咸味，以及和这些味道混在一起的某种植物油的香味，将这些滋味全部混合在一起的复杂酱汁如果单吃会显得味道太强烈，但搭配肉汁与充满油脂的肉，就成了极为契合的暴力滋味。

虽然这种风味和宫廷流行的「优雅料理」实在是天差地远，但也因此紧紧抓住了菲丽的内心。

(接下来才是牛五花盖饭的重头戏！)

不过现在还不能松懈，不只是肉。单靠肉无法完成牛五花盖饭。既然已经开始行动，菲丽不再有任何迷惘。为了享受更进一步的滋味，她将筷子插进碗里。

她挖了一口因为肉被夹走而探出头的米饭，放进嘴里！

(――――！)

菲丽的脑中已经无法浮现任何词汇，比海国的米还要甘甜的香味与滋味在嘴里扩散开来。煮得软嫩的米饭，逐渐和蹂躏嘴里的肉混合在一起，最后产生的，就是让菲丽沉醉不已的极致滋味。

这个味道迷惑了菲丽，让「海国高贵的公主」摇身一变，成为一名「正在发育的年轻少女」。

不只是味觉，在享受甜辣的味道、颜色鲜艳的蔬菜与肉和米的对比，从手掌传来的碗的热度，以及从胃里涌出的食欲的同时，菲丽一点一点地慢慢吃完眼前的料理。

她慎重地吃肉和吃饭，以免弄脏身上的衣服，偶尔换换口味吃些深绿色的凤草、带有些许甜味的卡琉特，以及口感清脆的森贤人豆芽、这些配菜都只有稍微用盐调味过，本身的味道非常清淡……不过这样才能让饭与肉的暴力滋味维持新鲜，所以再也没什么比这些配菜更适合转换口味了。

接下来，她将肉和饭搭配蔬菜一起吃。

原本完成度就很高的肉和饭，在加上新的蔬菜味后，又让牛五花盖饭展现出不同的风味，肉汁、调味酱汁和口感绝佳的蔬菜，这样的组合同样让人欲罢不能。

等吃了几口牛五花盖饭后，再直接以碗就口喝一点加了海藻的汤，用与牛五花盖饭不同的味道让舌头休息……重新调适过的心情，将化为继续吃牛五花盖饭的原动力。

菲丽已经创不出话。

因为她将舌头完全用在品尝味道上。

于是令人眼花瞭乱的味道竞演持续不断地展开。

过不久，那场竞演终于结束。

「――呼。」

即使花了比一般客人还要久一点的时间，菲丽仍顺利吃完了牛五花盖饭，最后她享用著加了冰块的凉茶，吐了口带有肉的香味的气息，将餐点全部吃完后，只剩下压倒性的饱足感，从末自平常那些无味的餐点感受到的饱足感，让菲丽觉得满足，甚至还感到幸福。

(果然牛五花盖饭才是异世界最棒的料理―― )

就在她边想边松懈下来时，刚才的服务生又走了过来。

「那个，我端甜点的霜淇淋过来了。请让我收拾一下餐具。」

「嗯，拜托你了！ 」

收拾完那些被吃得乾乾净净的餐具后。魔族服务生换摆了一个装著又白又软的冰凉霜淇淋的器皿到桌上。

「……喔，好甜……」

刚才享用牛五花盖饭的过程，就算说是一场战斗也不为过。而浓厚的冰凉甜味，正好适合为这餐做收尾，菲丽像吃牛五花盖饭时那样'，―匙―匙地慢慢吃著霜淇淋，仔细品尝。

就在原本装满霜淇淋的器皿被清空时――

「呼……吾满足了。那么吾差不多该回去了。」

对店里的服务生这么说后，菲丽轻轻将饭钱放到桌上。

那是菲丽本人除了在这里以外，根本没机会用到的银币，她拜托女官让给自己几枚银币，但今天只用到一枚，没错，牛五花盖饭、茶和霜淇淋，这些料理加起来只要―枚银币。

(――真的只付这些就够了吗？)

虽然菲丽每次都会这么想，但她知道店长绝对不会收超过这个金额的钱，所以轻轻放下银币。这是菲丽给这间店的小小回报，考虑到那美妙的味道，这实在是远远低于行情。

「好的！谢谢惠顾！」

看了收下银币后便开始收拾桌面的服务生一眼后，菲丽离开店内。

――一口气吃完牛五花盖饭后，就该用霜淇淋收尾。吃完这些东西后，就该立刻离开。

菲丽遵循过去带她来这间店的旅小人的教诲，返回位于宫廷中庭的林子。

(……今天已经不用吃午餐了……)

菲丽摸著肚子走回自己的房间换衣服，同时如此想著。

虽然现在已经差不多是午餐时间，但她完全没心情吃其他料理。

就用身体不舒服当藉口吧。

都已经这么饱了，再吃其他东西也只会破坏兴致。

在太阳爬到天空中央时，菲丽暗暗下定决心，一只乌鸦凝视著菲丽……它是被主人赋予守护公主「冒险」的任务，并基于这个目的创造出来的式神。

「毕竟如果不偶尔让公主透透气，她迟早会被闷坏呢。」

直接和式神共享视觉的道俊，在前往赴宴的马车上晃著身了如此低喃。
