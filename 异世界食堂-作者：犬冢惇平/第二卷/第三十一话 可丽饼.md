<p align="center">第三十一话 可丽饼</p>

东大陆有片至今仍人迹罕至的原野，那里有个被称为常花之国或是花之国的小国。

在那个冬长夏短的地区里，就只有那个小小的国家有片全年都是温暖春天的花田。

基于那样的地理性质，在约百年前的「邪神战争」中，各方势力都想攻下那里建立前线基地，不论是人类的联合骑士团，还是虽然身为魔族但仍不断进行钻研与探求最后成功转生为魔灵，使用凶恶的魔法受人恐惧的「魔骸王」率领的魔族们，都曾经侵略过这个花之国。两方人马都为了决战而准备了充足的装备与人数，不过企图侵入这片花田的他们……最后全都被这个国家的「国民」袭击，在失去了大部分的战力后凄惨地战败。

没错，那个全年都盛开著各色花朵的国家，从一千多年前开始就一直是拥有翅膀的小人们――「妖精」的国家。

妖精。这个种族拥有能坐在人类手掌上的娇小身体、自由在天空飞翔的蝴蝶翅膀，以及能和精灵匹敌、完全足以弥补肉体方面无力的魔力。因为能使用魔法，从整个世界的角度来看，他们绝对不算是个弱小的种族，有时候甚至会被人类冒险者或魔物恐惧，但他们个性温厚，不会利用那股魔力侵略其他地方，同时也是一群不允许他人践踏自己国家的勇士。

因为这样的个性，当时被人类与魔族任性的入侵激怒的女王，希尔薇雅．席尔巴里欧十三世亲自率领精锐，利用地利与有限的魔力勇敢应战，漂亮地击退了敌人。在那之后，人类与魔族就将那片花田视为擅入就会死亡的禁地恐惧，花之国也在没有发挥那股力量的情况下享受了漫长的和平。

那阵和平的停滞开始出现改变的迹象，是在约半年前的时候。

「陛下！大事不好了！」

那道声音在某天中午，响彻了城墙是由各色花朵和嫩绿新芽建立而成的城堡。

「嗯，是和之前那个东西有关吧。」

相对于急忙冲进来的高龄魔法师团长，拥有雪白肌肤与黄绿色的眼睛和头发，以及只有边缘是黑色的虹色美丽翅膀，在两年前继承了因病去世的母亲的地位，成为现任花之园女王的蒂亚娜。席尔巴里欧十六世厌烦地回答。

「……陛下也发现了吗？」

这句话，让魔法师团长忍不住反问道，蒂亚娜用力点头回答

「当然。这里是吾等的庭园。要是发生什么异变，吾马上就会知道何况还是在吾等城堡的正前方。」

当天，从开始处理公务后就一直没离开这个王座大厅的蒂亚娜，利用她那无疑是全国最为优异的魔法师才能，感觉到了刚才发生的异变。

妖精因为其优异的魔力和无力的身躯，非常重视魔法。他们战斗时不像人类那样用剑或弓，主要都是使用魔法，或是用魔法产生的魔像，因此自古以来，在选择统率妖精的王时都非常看重魔力，不论男女，只会选择魔力优秀的人。所以身为妖精之长的女王必然也是位优秀的魔法师。

「不能放著不管，先去视察看看吧。」

蒂亚娜像是原本就在等待这个报告般起身。

「无论那是什么，吾都不容许任何事物践踏吾等的领土，走吧，各位，跟上来！」

女王呼唤默默在一旁守护的近卫们，飞上天空。没错，这里是花之国，绝对不容许任何人玷污在园内盛开的花朵，骄傲的妖精之园。周围的骑士和大臣们都点头肯定女王的话，默默地跟随在后，然后，他们来倒了花之国出现在那座看在人类眼里非常小的城堡前方的异变面前。

「……这是……门吗？」

与好奇地聚集在这里的民众们一起观看那扇叫的蒂亚娜，忍不住如此低喃，那是人类使用的大门……花之国内原本没有那种门。上面绘有某种野兽的图案，附带金色门把的黑门。当然，昨晚还没有这个东西。

「……是某种召唤魔法吧……好。」

因为完全看不出这东西的真面目，蒂亚娜下定某个决心。

「接下来要进入这扇门，从魔法师团里选几个人留下来守卫本国，剩下的人跟吾一起过去。」

只要能确认这扇门的真面目，国民们就不会产生不必要的不安，蒂亚娜快速做出决断。

「什么？陛下！这样太危险了吧？」

「别小看人，吾可是这个国家的女王，吾有责任守护国家不用担心，吾会先准备好守护国家的战力再出发。」

安抚完惊讶的魔法师团长后，蒂亚娜使用了花之国的秘密武器。

「守护者啊，回应吾的呼唤，出现在吾面前吧！」

她发动魔法，从怀里的袋子里取出一粒种子扔到地上，那粒种子来自据说早在精灵自诩为世界支配者前的「古龙时代」就已经存在的花，而且一年只能采集一次种子，即使在这个花之国中，那种花也特别被城堡的魔法守护，并只能在唯有宫廷的园丁和历代女王能照顾的特别花坛中培育。

藉由施展只传给历代女王的秘术，花之国最强的守护者诞生了。

喔喔喔喔喔……

伴随著彷佛从地底响起的声音，种子吸收周围大地的恩惠，急速成长，从种子涌出深绿色的藤蔓，那些藤蔓伸长后互相缠绕，逐渐产生形体，最后出现一个绿色巨人，藤蔓就像肌肉般集结在一起，看似眼睛与鼻子的地方则是开出桃色的花，巨人高到蒂亚娜等人必须仰望……大约有人类的三倍高。

「很好，守护者啊，听从吾的命令。」

喔喔喔喔……

巨人以咆哮回应女王的命令，认其为自己的主人。

「很好……国民们，先离这扇门远一点！」

蒂亚娜等人留下巨人，拉开与门的距雕。目前只知道那扇门是魔法的产物，所以无法预测隐藏什么样的危险。蒂亚娜基于此做出这样的判断。

「很好……守护者啊！打开那扇门！」

喔喔喔喔喔……

巨人遵从女王的命令弯下身子，灵巧地用由祖树枝构成的两根手指握住门把，轻轻转动。

叮铃铃……

伴随著象徵魔法发动的铃声，门应声开启。

「……喔，这是转移装置啊。」

虽然对面的景色模湖无法看清，但这是能看见有几个像人类的物体聚集在一个明亮的房间内。蒂亚娜知道这是一种使用转移魔法的魔法装置。

「陛下，请您指示？」

「……这个嘛。首先……」

魔法师团长的话让蒂亚娜还视周围，看向花之国的精锐魔法师们。

「去打探对方的意图吧，由吾亲自指挥部队。没意见吧？」

做出决定后，接下来就简单了。蒂亚娜迅速做出决断，然后下达指示。

「这怎么行……这样太危险了！都还不晓得对面有什么东西！而且这扇门的尺寸还不足以让巨人通过喔？」

「这不算什么，吾会先做好准备。魔法师团长留在这里，准备撤退用的魔法阵，要发生状况时，能够马上发动的那种。」

蒂亚娜无视魔法师团长的警告，长年维持花之国的独立，持续保护领土的历代女王，在身为女王的同时也是妖精中最强的魔法师，比起留在安全的场所拟定战略，她们更尊崇亲自上阵的勇者，而蒂亚娜也同样要求自己必须当个不辱这项传统的女王。

「这是用来招待吾等的东西，既然如此，那就顺对方的意吧。」

仔细想想，自从那些被花之国认为无耻的人类和魔族进来这里以来，其实已经有一百年没发生过像这样的异常状况，这件事实让年轻女王在不知不觉间感到兴奋，没错，虽然之前因为太过和平而没机会发挥实力，但现在正是展示女王的知识与魔法的时候。

「吾乃统率本国臣民者，所以有必要和对面的首领沟通。」

即使可能伴随危险，她也没打算退让，女正意气风发地用飞的穿过门。

一抵达门的对面，原本模糊的光景就突然变得清晰。

「这，这里是什么地方？」

其中一名随行的魔法师慌张地问道。那里就是如此异常的空间。

「……冷静点。」

出现在眼前的光景 即使人类、魔族，亚人和魔物都不是在同一个场所战斗，而是各自吃著自己的料理，惊讶的女土还是指示部下们不要慌张。

(这里到底是什么地方…… )

当然，蒂亚娜自己也不晓得这里是哪里，目前的情报太少了。为了尽可能搜集情报，蒂亚娜集中精神观察魔力。

「……那个人是半精灵的魔法师吗？」

然后蒂亚娜盯上了其中一个人。穿著设计简单但作工精致的蓝色洋装的银发半精灵，从那位半精灵将轻型的秘银制魔杖放在一旁来看，她应该是魔法师。此外从魔力的强度和对魔力的完美控制来看，那个人应该天生就拥有足以和蒂亚娜匹敌的高强魔力，并在某位良师底下经过相当的磨练。

(那个人应该知道这里是哪里。 )

如此判断的蒂亚娜对部下使了个眼色，然后走向那位魔法师。

「……这位小姐。您看起来似乎是名魔法师。吾乃花之国的女王蒂亚娜．席尔巴里欧十六世。魔法师大人，若您不介意，可以告诉吾这里是什么地方吗？」

女王的话让半精灵魔法师停下了动作，她原本在吃一种像黄色史莱姆的料理。

「我是萨玛纳克公国王女，维多丽亚．萨玛纳克。花之国的女王，蒂亚娜陛下，能见到您是我的荣幸……这里是异世界食堂。异世界的『猫餐厅』与我们的世界透过那扇门相连，是个用餐的场所。」

维多丽亚简洁地说明异世界食堂是个什么样的地方。

「异世界食堂？用餐的场所？」

「没错。这里能用人类的钱购买料理……亦即有经过切碎或用火煮过的食物，这里有许多在我们的世界吃不到的珍奇料理，所以有许多客人慕名而来，我也是其中一人。」

从维多丽亚那里听到关于这间店的大致说明后，蒂亚娜陷入思考。

「提供用火煮过的食物的店……这么说来，吾有听创过人类和精灵喜欢那样的东西……」

蒂亚娜想起小时候从曾去人类世界旅行过的教师那里听来的知识，人类和直接吃花田里长出的果实和花蜜的妖精不同，他们似乎会用火烤食物，或是用热水煮东西来吃，而异世界食堂是个提供经过各种加工的食物的店。

「不过……做这种事情会让东西变好吃吗？根据吾从曾去人类世界旅行过的人那里听来的话，似乎没有花之国的花蜜那么美味。」

「很好吃。」

维多丽亚信心十足地点头回答蒂亚娜的疑问。维多丽亚持续光顾这间店已经八年了。这间店的料理好吃到让她直到现在都还吃不腻。

「这样啊……凡事都需要经验，不好意思，魔法师大人，能向您请教有没有什么比较符合吾等口味的料理吗？」

看维多丽亚回答得如此肯定，开始对这里产生兴趣的蒂亚娜向维多丽亚问道。

「我知道了。适合各位的料哩吗……

另一方面，维多丽亚在听见蒂亚娜的问题后陷入思考。

(首先推荐甜点准没错。)

妖精的主食是花蜜和果实。两者都是甜的东西。

(……圣代的容器是个问题。)

维多丽亚瞄了一眼正满脸笑容地吃著圣代的皇女……舍弃这个选项，装圣代的容器很深，让这些小妖精们用比自己身高还深的容器吃东西太辛苦了。

(法式布了……妖精讨厌苦的东西，所以不行……)

接著始瞄了一眼自己正在吃的某种装在浅器皿上的东西，遣憾的是这个也不适合。维多丽亚认为没有焦糖就不算是布了。

(应该还是包含许多水果的甜点比较好，考虑到最好是平常吃不到的东四，就要找不是生的，而是加工过的东西。这样一来……)

然后，维多丽亚终于想刻了答案。

「……综合水果可丽饼，我推荐你们点这个。」

「这样啊，那就这个吧。」

蒂亚娜大方地接受了维多丽亚的提议。

「我知道了……店长。请给我们一份综合水果可丽饼，最好能切得像磅蛋糕那么小块。」

「好，好的。」

虽然店长因为初次见到宛如从童话故事里跑出来的妖精而吓了一跳，但还是帮维多丽亚点菜，在回过神后返回厨房。过了不久，金发的魔族少女就将放在盘子上的「那个」端了过来。

「久、久等了，这是综合水果可丽饼。」

不晓得是因为不习惯服侍像蒂亚娜这样身分高贵的人，还是对一脸警戒的妖精们感到困惑而有点颤抖的手，将某种掺杂著淡黄色与褐色花纹、像布一般的东西放到桌上，那东西像花束般包著白色的柔软物体与五颜六色的水果。

(喔。这就是可乐饼啊……人类也满会想的嘛。)

红、橙、绿……包覆著各种新鲜水果的纯白色物体与带有美丽花纹的布。可乐饼是种外观看起来非常漂亮的食物。

「那我用这个帮各位切一下。」

之后魔族少女为了让眼前的小人们方便吃可乐饼，将可丽饼切成一口大小、她用保养良好的银色刀具，将可丽饼切碎，并按照店长的指示，尽可能将每一块都切成相同的大小……小心不要做出没包含水果的「坏签」

「那么请各位慢用。」

全部切好后似乎松了口气的魔族少女说完后，就回厨房拿其他客人点的料理。

「……那么，就从吾先开始吃吧。」

在被留在桌上的一群人中，蒂亚娜如此低喃。

「这怎么行？说不定有毒！太危险了！」

「所以才应该这么做。 。」

蒂亚娜冷静地回答

「在吾的魔法加护面前，不管什么毒都没有意义。既然如此，当然应该由吾先吃。」

虽然蒂亚娜冷静驳斥部下，但这只是藉口。坦白讲……她非常好奇眼前的料理吃起来是什么味道。

(首先是……嗯？几乎没什么味道呢。 )

蒂亚娜先拿起离自己最近的一块，她咀嚼著看似布角的部分，在心里感到困惑，这个部分几乎没味道，妖精通常喜欢甜的东西，所以这味道让蒂亚娜觉得有点扫兴。

(嗯……接下来是这个白色的东四……唔？)

稍微降低了期待标准的蒂亚娜，在舔了那个白色物体后惊讶地睁大眼睛，彷佛入口即化般的甘甜与柔软，虽然甜，但不仅仅是甜而已，还伴随著新鲜牛奶的味道。

(太棒了！这个甜味……唔唔嗯？)

就这样继续吃下去的蒂亚娜，在邂逅了那个后睁大眼睛，那是种橙色的水果，又甜又充满水分，直接吃也非常美味。

(这是……将甜的水果进一步泡在更甜的水里吗？ )

原本就比蒂亚娜所知的水果还要甜的水果上！还加了甜甜的蜜汁。这让可丽饼内的水果变得宛如会在嘴里融化般甘甜，让蒂亚娜觉得像是在作梦一样。

〔太美味了……该不会？ )

享受完这个味道后，蒂亚娜察觉到一项事实，看向可丽饼……看向还包了许多和刚才的橙色物体不同的各种水果的可丽饼。

(果然！)

然后吃过那些东西的蒂亚娜，发现每一样水果都被下了不同的工夫。

红色莓果事先加了大量砂糖炖煮，营造出黏稠又滑顺的甘甜风味。

透明的绿色莓果似乎事先用某种魔法冷冻过，带有冰凉的爽脆口感。

然后是连蒂亚娜都不太清楚，看起来是从某个环状物体切一小块下来的黄色水果，尽管那种水果原本就带有一定程度的酸味，但又包含了和橙色水果一样甘甜的蜜汁。

(原来如此，包覆这些东西的布是为了这个……)

然后在吃这些东西的过程中，蒂亚娜发现刚才被她断定没味道的布，其实也是经过计算的产物。支撑那个又白又甜的物体，以及各种带有强烈甜味的水果配料的基础，就是这块布，刻意不对这块布进行调味，是为了用来调整太甜容易让人吃腻的可乐饼的味道。没错，就是因为有这块布，可丽饼这道料理才得以成立。

――啊啊，这个可乐饼果然是毒……

大口吃著味道完整的可丽饼，蒂亚娜领悟到了这点。明明肚子已经很饱。但手还是停不下来，不断将可丽饼往胃里送。

―――这一定是用来让吾等的肚子破裂的毒。

甚至产生了这样的想法。酸甜水果与白色甜食的诱惑实在太甜美。不管再怎么强大的魔法防护，对「吃太多」都没有抵抗力。直到证验了有生以来最饱的经验后，蒂亚娜才总算停了下来。

蒂亚娜一直、一直挑战自己的极限猛吃可丽饼，直到打了一个可爱的嗝后才总算停止。然后她叹了口甜甜的气，转头看向跟随她来到这里的部下。

「……陛下，那东西吃起来究竟如何……」

一名亲信的魔法师咽了一下口水，向蒂亚娜问道。对此，蒂亚娜擦掉嘴角的鲜奶油后，只回答一句话。

「嗯，没问题……可以吃。」

蒂亚娜一说出这句话，妖精们便冲向可腿饼。女王之前在他们面前吃得津津有味，让所有人都再也无法忍耐，在实际吃过后，更是为那彷佛在嘴里融化般的甜味发出欢呼。

「……维多丽亚，感谢您，这是谢礼，请收下吧。」

看著那样的部下，蒂亚娜将符合王族身分的奖赏花之国的秘宝交给维多丽亚。

「这是在吾等的国家采的花的种子……如果是您，应该能理解这代表什么意义吧？」

「……没关系吗？」

乍看之下是随处可见的褐色种子，但维多丽亚从里面蕴含的惊人魔力看出那个种子的真面目。在距今非常遥远的古王国时代，当时的王为了廷长寿命而不断追求，最后修于绝迹的梦幻花朵的种子。光是直接煮来喝就能回春―岁，虽然也能当成强力的魔法触媒，但由于只能种在包含莫大魔力的土地，因此据说现在绝对不可能开花，就连公国都只有―粒，而且还被放在施了严密的魔法保护的宝物库里。的确是足以称作秘宝的东西。

「放心吧，这不成问题，虽然听说这对人类来说是极度贵重的物品，但吾也收到了相对应的东西。」

蒂亚娜对维多丽亚露出比刚才更加妩媚的笑容说道。

「……我知道了。那么从今以后，各位在这里吃的料理费用，全都由我来支付！」

为了回应收到的贵重礼物，维多丽亚对蒂亚娜立下盟约。即使必须替妖精们付钱到自己的寿命终止为止，和这粒种子的代价相比还是太便宜了，不过维多丽亚也想不到自己还能替妖精女王做些什么。

「这样啊。其不好意思。」

最后蒂亚娜也坦率地对此表达谢意。与人类社会隔绝的花之园，不容易取得人类的货币。蒂娅娜原本就在担心以后要如何支付可丽饼的对价，所以坦率地对这个解决方法感到高兴。

……在她的心里，早已决定之后每隔七天都要来这里一次。

(反正其他人应该也不会反对。 )

多达百名的部下已经吃光那块巨大的可乐饼，而且还希望「再来一份」。蒂亚娜完全不认为他们会反对以后再来这间店。

在那之后过了半年，今天的「门对策会议」也产生了纠纷。

「今天应该点巧克力香蕉！你们难道不懂那种苦味的好吗！」

「你在说什么啊！当然是甜甜的草莓果酱和酸甜的奶油起士的绝佳组合比较好吧？」

「如果只看甜味，那当然是卡士达酱比较好！应该挑以卡士达酱为主的口味！」

「不对！错得离谱！卡士达酱激不出鲜奶油那种柔顺的口感！鲜奶油比卡士达酱好吃，这才是世界的真理！」

「那个~听说可乐饼也有不甜的口味。」

「「「驳回！不甜的可丽饼根本就不是可丽饼！」」」

议题是「除了综合水果以外还要点哪种口味」，但完全无法取得共识。

「陛下！这次被选上的国民已经在猫餐厅们的广场集合！如果不尽快出发，可能会发生暴动！」

没入选会议的魔法师团长进来向蒂亚娜报告。

「吾知道了，吾立刻过去！」

蒂亚娜用力点头回应，然后缓缓起身。

「出发了，虽然很遗憾没有结论，但剩下的就到那里再考虑吧！」

说完后，蒂亚娜前往国民们都在等待伟大女王现身的广场。

在那之后，将情报带回去的蒂亚娜等人在经过反覆讨论后，针对猫餐厅订了一条规定。即使是人类用的巨大店面，要是随便就超过几千名的国民全部一起涌进去，还是会给店家添麻烦。因此他们订下了一条规定，那就是无关地位、年龄或力量，只有经过严格抽选出来的两百名人民，能和蒂亚娜一起通过那扇每七天只会开一次的门。

「国民们！让各位久等了！现在就出发吧！前往异世界食堂！」

蒂亚娜对带著用来装异世界点心的用藤蔓编成的笼子，毫不隐藏期待地飞在空中的国民们喊道，国民们发出欢呼，聚集到门的前面。

「很好！魔像啊！开门吧！打开通往异世界的道路！」

收到命令后，最近已经彻底变成开门工的巨人握住门把，将门打开，伴随著来客铃的铃声，无数的妖精穿过那扇门。

「啊！欢迎光临！」

最近被雇用的魔族服务生见状，便熟练地笑著说道。

「嗯。先帮吾等带位。然后再拿菜单过来。」

对那位服务生点头示意后，蒂亚娜威风地下令。那一天也同样展开了一场甜美的宴会。
