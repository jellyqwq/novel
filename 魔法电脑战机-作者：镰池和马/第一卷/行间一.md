<p align="center">行间一</p>

大气组成和重力值基本相同，但元素周期表不完全相符。

没有earth水晶的反应虽然让人惊讶，但这也是可能性之一。

有着饱满的众行星和没被限定战争茶毒且无限扩张的宇宙，只要对若干的砕片睁一只眼闭一只眼就能感受到它的清新无垢，用望远镜窥见的每一颗星星仿佛都是梦想之地。这仿佛是在直视不知自己深陷危机的无防备的少女的眼瞳，让我不禁涌现出罪恶感。

白金之芽以完全预料之外的形式存在，实在令人震惊。

虽然压缩和整形花了不少功夫，但实在让人觉得有趣且充实。果然我拥有着并非军人而是学者的灵魂。

难题堵在眼前的时候实在令人感到幸福。真正悲剧的是即使翻遍整个火星沙漠也找不到一颗沙金，可能性全灭的状态。这毫无疑问使我体验到的名为人生的事物。

我和其他众多的同志们一样，没被Tangram选上。

但即使没被选上，也存在着没被选上的用途。

自豪吧，选择者。直至推翻自身所受的惩罚，贯穿那只『眼睛』。

Code Phoenix已经平安度过了第一阶段。

和步入众多『道路』的同志一起切入第二阶段吧。

一切都是为了引导至第三阶段的终结。
