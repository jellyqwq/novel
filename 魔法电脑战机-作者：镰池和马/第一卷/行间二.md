<p align="center">行间二</p>

Code Phoenix出现超越第二阶段的兆头。

白金之芽与真正被Tangram所爱的女人在言行上有着巨大的差别，但这也是因为具有无数分支可能性所致。

人类的NPC化是预料之外的变质，但根绝Shadow化这一不同的可能性来看着其拥有可采用的部分。

实际上，我等没有突破第三阶段的胜算。

只是首先入手名为白金的世界最大级黑匣子，然后凭它的力量得到第三阶段的解决方法。

我等是何等的羞耻。不仅不断抱怨着逃离不了白金的束缚，却仍将白金神格化。这不就和将动物学者污蔑成操纵使魔的魔女之辈一样吗。

我们的栖身之地已经堵塞。一切都被数值化，一切都可进行演算，是个可能性之轮被封闭的世界。数个肥大的骄傲完全掌握了财富，将下克上之芽完全封杀的冰冷牢狱。

其中最有象征性的存在就是屹立于V水晶、电脑虚数空间、Virtualroid、原版飞燕，以及Tangram等超科技前方的一名研究员的名字。一切科技就是那个名字，被那个名字所选中的存在就会受到科技的恩惠，成为人生赢家。那就是被这样神格化和神圣化的开发者。

莉琳·普拉基纳。

粉碎那少女外形的世界天枰，将可能性带回世界。为此，我们利用Tangram的拒绝反应来到了其它道路。

即使我们需要依靠另一个白金·富良科凛铃，也要获得成功。

来吧，最后的游戏开始了。
